Show solutions

Proper function: A proper function \( f: X \to Y \) is a relation from \( X \) to \( Y \) such that \[ \forall x \in X, \exists! \, y \in Y \text{ s.t. } (x, y) \in f \] (i.e., each input has exactly one output). It can be one-to-one or many-to-one. Note: a proper function might still be ambiguous if the rule defining it is inconsistent.

(a) Arbitrary assignment: An arbitrary assignment assigns outputs to elements of a set without necessarily obeying the rules of a well-defined function. Observation: Being ambiguous is logically equivalent to being not well-defined.

(c) Why every domain element must have an image: If any domain element had no image, the assignment would be incomplete or ambiguous. A well-defined function must assign exactly one output to each domain element.

Two functions \( f, g: X \to Y \) are considered equivalent if \[ \forall x \in X, \ f(x) = g(x). \] - This is the **mathematical definition**: outputs match for all inputs. - In everyday language, “equivalent” can just mean “similar” or “interchangeable,” but in math it is **strict and precise**.

7: Topology and Metrisability

Show solutions

(a) Why impose a topology?
We impose a topology on elements of vector or function spaces to define notions of **closeness, limits, continuity, and convergence**. Topology formalises which elements are "near" each other without necessarily using numbers.

(b) Definition of metrisable space:
A topological space \( (X, \mathcal{T}) \) is metrisable if there exists a metric \( d: X \times X \to [0, \infty) \) such that the topology induced by \( d \) is exactly \( \mathcal{T} \). Formally: \[ \mathcal{T} = \{ U \subseteq X \mid \forall x \in U, \exists \varepsilon > 0 \text{ s.t. } B_d(x, \varepsilon) \subseteq U \}, \] where \( B_d(x, \varepsilon) = \{ y \in X \mid d(x, y) < \varepsilon \} \) is the open ball of radius \( \varepsilon \) around \( x \). In words: a space is metrisable if we can assign a distance function such that "closeness" and "open sets" in the topology are exactly described by that metric.

Key Point: Vectors as functions

If an element is both a vector and a function, we can describe a **finite vector** in \( \mathbb{R}^n \), say \((y_1, y_2, \dots, y_n)\), as a function \[ f: \{1, 2, \dots, n\} \to \mathbb{R} \] where \( f(i) = y_i \). This gives the list of components that form the vector.

For finite vectors, this function viewpoint is **possible but not necessary**. For **infinite-dimensional vectors**, the domain (index set) is infinite, e.g., \( \mathbb{N} = \{1, 2, 3, \dots\} \), and so the **function understanding becomes essential** for analysis.

This concept explains why function-based perspectives are fundamental in functional analysis, especially for infinite-dimensional spaces.

Key Point: Sequences, infinite-dimensional vectors, and functional perspective

A sequence is an infinite list of elements indexed by the natural numbers \( \mathbb{N} = \{1, 2, 3, \dots\} \). Even if the outputs are simple, like \(1, 0, 0, 1, 0, \dots\), there are infinitely many values.

A sequence can describe an infinite-dimensional vector, where each element of the sequence corresponds to a component of the vector.

The sequence space is the set of different sequences. Each sequence is infinite and describes a single vector.

To construct an infinite-dimensional vector, we need an infinite set of basis components. - If we have a vector that is "one-to-one," only one basis vector is needed to describe that vector. - To describe all vectors in an infinite-dimensional vector space, we require **infinitely many basis vectors**.

Remember: these vectors can also be viewed as **functions**, where the domain is the index set (natural numbers) and the output is the component value. This is why **functional understanding is essential** in infinite-dimensional vector spaces.

Key Point: Norms, metrics, and infinite-dimensional vectors/functions

In finite-dimensional vector spaces, we often talk about **weights, lengths, distances, and angles** between vectors. In many manifolds or abstract spaces, we want the same: to measure **distances and angles** between vectors or functions.

A norm is a function that assigns a size (length) to each vector or function: \[ \|\cdot\| : V \to [0, \infty) \] and satisfies positive definiteness, homogeneity, and the triangle inequality.

A norm induces a metric via \[ d(u,v) = \|u-v\|. \] This metric tells us the "distance" between any two elements.

Why not just define a metric directly?
- A metric measures distances, but does not automatically give you **lengths, scaling, or angles**. - A norm gives **more structure**: it tells you both the size of individual elements and induces a metric for distances. - Especially in infinite-dimensional spaces (e.g., sequences, function spaces), using a norm allows you to talk rigorously about:

  • Distances between elements
  • Cauchy sequences and convergence
  • Completeness of the space
  • Linear structure (scaling and addition of vectors/functions)

So, the norm is more than just a metric: it preserves the **linear and functional structure** of the space while giving a natural metric to define limits, convergence, and continuity.

Key distinction:
- In **functional analysis / infinite-dimensional vector spaces**, we use norms because we need both **linear structure and distance**. - In **general relativity / manifolds**, the space is **not a vector space**, so we only need a **metric tensor** to measure distances and angles; a separate norm is unnecessary.

Key Point: Closed subspaces and Cauchy sequences

A subset \(M \subset V\) of a normed space \(V\) is closed if it contains all its limit points: \[ \text{If } (x_n) \subset M \text{ and } x_n \to x \text{ in } V, \text{ then } x \in M. \]

Why closedness matters:

  • In infinite-dimensional spaces, the image of a linear map may not be closed, even if the map is linear and bounded.
  • If the image is not closed, a **Cauchy sequence in the image may converge to a limit outside the image**, breaking convergence within the subspace.
  • Closedness ensures that **limits of Cauchy sequences stay in the subspace**, which is essential for completeness and functional analysis constructions.

Summary: Closedness = all limit points included. Necessary so that Cauchy sequences in a subspace converge **inside** the subspace, especially in infinite-dimensional spaces.

Key Point: Completeness, norms, and convergence in normed spaces

A normed space \((V, \|\cdot\|)\) is complete if every Cauchy sequence in the space converges to a limit in the space.

Norm and distance:
- A norm assigns a size (length, or "weight") to each element, even if the element is an infinite-dimensional vector or a function. - The norm induces a metric: \[ d(u,v) = \|u-v\|, \] which allows us to measure the distance between elements in a precise way.

Cauchy sequences:
- A Cauchy sequence \((v_n)\) satisfies: \[ \forall \varepsilon > 0, \exists N \text{ s.t. } \|v_n - v_m\| < \varepsilon \quad \forall n,m \ge N \] - Not every sequence is Cauchy, but in a **complete normed space**, every Cauchy sequence converges to a limit within the space. - The norm ensures that the "size" of differences between sequence elements is measurable, so we can meaningfully talk about convergence, even for infinite-dimensional elements.

Important note on finiteness:
- The norm gives the "weight" of an element, but it does not guarantee the space is bounded; elements can have arbitrarily large norm. - What matters for convergence is that the sequence is Cauchy — distances between elements shrink — not that individual elements have finite norm.

Key Point: Complete normed spaces (Banach spaces)

A complete normed space, also called a Banach space, is a normed space \((V, \|\cdot\|)\) in which every Cauchy sequence converges to a limit within the space.

- The norm defines weights of each element and then the distances between elements: \(d(u,v) = \|u-v\|\). - Cauchy sequences are sequences whose elements get arbitrarily close. - Completeness ensures that these sequences have limits that stay in the space, making convergence meaningful even for infinite-dimensional elements.

Summary: Complete normed space = “all Cauchy sequences converge inside the space.”

Key Point: Inner product and Hilbert spaces

- A norm gives the size (length) of vectors/functions, but it does not define angles. - To define **angles, orthogonality, and projections**, we use an inner product: \[ \langle u, v \rangle \] which satisfies linearity, symmetry (or conjugate symmetry), and positive-definiteness.

- A **Hilbert space** is a complete normed space in which the norm is induced by an inner product: \[ \|v\| = \sqrt{\langle v, v \rangle}. \] - This allows us to rigorously define **lengths, angles, orthogonality, and projections** in infinite-dimensional spaces. A manifold is able to define the weight of a vector, and distances and angles between other vectors, solely using the metric. For infinite-dimensional spaces, we use the norm and inner product.

9: Projection onto a subspace

Define the projection of a vector or function onto a subspace. Give the key properties and intuition.

Show solution

Definition: The projection of a vector or function \(x\) onto a subspace \(M\) is the element \(y \in M\) such that the difference \(x - y\) is orthogonal to \(M\).

Formally, let \(H\) be a Hilbert space and \(M \subset H\) a subspace. For every \(x \in H\), the projection of \(x\) onto \(M\) is the unique \(y \in M\) such that: \[ x - y \perp M \] That is, \(\langle x - y, z \rangle = 0\) for all \(z \in M\).

Intuition: \(y\) is the closest point in \(M\) to \(x\), so the projection “drops” \(x\) onto the subspace along the shortest path.

Key Point: Banach function spaces

A Banach function space is a function space that is also a complete normed space. That is, it consists of functions with certain properties, equipped with a norm, such that every Cauchy sequence of functions converges to a function in the space.

Examples:

  • C([a,b]): the space of all continuous functions on the interval [a,b], with the supremum norm \(\|f\|_\infty = \sup_{x \in [a,b]} |f(x)|\).
  • Lp([a,b]): the space of Lebesgue p-integrable functions \(f\) such that \(\int_a^b |f(x)|^p dx < \infty\), with norm \(\|f\|_p = \left(\int_a^b |f(x)|^p dx\right)^{1/p}\).
  • lp: the space of sequences \((y_1, y_2, \dots)\) such that \(\sum_{n=1}^\infty |y_n|^p < \infty\). These sequences often represent the outputs of functions in a discretised sense.

Summary: Banach function spaces combine **function-specific properties** (like continuity or integrability) with **completeness under a norm**, allowing rigorous analysis of convergence and limits in infinite-dimensional spaces.

11: Linear maps, kernel, and image

Let \(T: V \to W\) be a linear map between vector spaces. Define the kernel and image of \(T\), and explain the relationship to injectivity and surjectivity. Discuss any special considerations for infinite-dimensional spaces.

Show solution

Definition of linear map:
\(T: V \to W\) is linear if \[ T(x + y) = T(x) + T(y), \quad T(\alpha x) = \alpha T(x), \quad \forall x,y \in V, \ \alpha \in \mathbb{R} \text{ or } \mathbb{C}. \]

Kernel:
The kernel of \(T\) is \[ \ker(T) = \{x \in V : T(x) = 0\}. \] Injectivity: \(T\) is injective if and only if \(\ker(T) = \{0\}\).

Image (range):
The image of \(T\) is \[ \operatorname{im}(T) = \{T(x) : x \in V\} \subset W. \] Surjectivity: \(T\) is surjective if \(\operatorname{im}(T) = W\). - In infinite-dimensional spaces, the image may **not be closed**, so surjectivity is harder to check. - Continuity (or boundedness) of \(T\) is often necessary to guarantee nice properties of the image.

Key Point: Dual Space V*

The dual space of a vector space \(V\) over a field \(\mathbb{F}\) is the set of all linear functionals from \(V\) to \(\mathbb{F}\): \[ V^* = \{ f : V \to \mathbb{F} \mid f \text{ is linear} \}. \]

A linear functional \(f\) satisfies, for all \(x, y \in V\) and \(\alpha \in \mathbb{F}\): \[ f(x + y) = f(x) + f(y), \quad f(\alpha x) = \alpha f(x). \]

The dual space \(V^*\) itself is a vector space under pointwise addition and scalar multiplication: \[ (f + g)(x) = f(x) + g(x), \quad (\alpha f)(x) = \alpha f(x), \quad \forall x \in V. \]

Intuition: Each element of \(V^*\) “measures” vectors in \(V\) by assigning a scalar. The dual space collects all such linear measurements, providing a powerful tool in functional analysis for understanding the structure of \(V\).

12: Dual space of a vector space

Define the dual space \(V^*\) of a vector space \(V\). Explain why in infinite-dimensional spaces, the dimension of \(V^*\) is strictly greater than the dimension of \(V\).

Show solution

Definition:
The dual space of a vector space \(V\) over a field \(\mathbb{F}\) is \[ V^* = \{ f : V \to \mathbb{F} \mid f \text{ is linear} \}. \] Elements of \(V^*\) are called linear functionals.

Infinite-dimensional case:
- If \(V\) is infinite-dimensional, there exist **more linear functionals than vectors in \(V\)**. - Intuition / sketch of proof: 1. Let \(\{e_1, e_2, \dots\}\) be a basis for \(V\). 2. Any sequence \((a_1, a_2, \dots)\) of scalars defines a linear functional \(f\) by \[ f\Big(\sum_i x_i e_i\Big) = \sum_i a_i x_i \] which is well-defined if only finitely many \(x_i\) are non-zero (Hamel basis). 3. There are infinitely many choices of sequences \((a_1, a_2, \dots)\), so **dim \(V^* >\) dim \(V\)**. - In contrast, in finite-dimensional spaces, \(\dim V^* = \dim V\).

Key Point: Topological Space

A topological space is a pair \((X, \tau)\) where:

  • \(X\) is a set (the underlying space).
  • \(\tau\) is a collection of subsets of \(X\) called open sets, satisfying the following axioms:

  1. The empty set and the whole set are open: \(\emptyset \in \tau\) and \(X \in \tau\).
  2. Arbitrary unions of open sets are open: \(\bigcup_{\alpha \in A} U_\alpha \in \tau\) for any collection \(\{U_\alpha\}_{\alpha \in A} \subset \tau\).
  3. Finite intersections of open sets are open: \(\bigcap_{i=1}^n U_i \in \tau\) for any finite collection \(U_1, \dots, U_n \in \tau\).

Summary: A topological space \((X, \tau)\) gives a notion of "openness" on \(X\), allowing us to define concepts like **continuity, convergence, connectedness, and compactness** in a general setting.

Key Point: Metric Space

A metric space is a pair \((X, d)\) where:

  • \(X\) is a set (the space of points or elements).
  • \(d: X \times X \to [0, \infty)\) is a function called a metric or distance, satisfying the following axioms for all \(x, y, z \in X\):

  1. Non-negativity: \(d(x,y) \ge 0\), and \(d(x,y) = 0\) if and only if \(x = y\).
  2. Symmetry: \(d(x,y) = d(y,x)\).
  3. Triangle inequality: \(d(x,z) \le d(x,y) + d(y,z)\).

Example: Normed spaces
A normed space \((V, \|\cdot\|)\) is also a metric space with \[ d(x,y) = \|x - y\|. \] This metric induces the topology of the space via open balls: \[ B(x, r) = \{y \in V : \|x - y\| < r\}. \]

Summary: Metric spaces formalise the concept of distance, allowing definitions of **convergence, Cauchy sequences, continuity, and compactness** in a general setting. Norms automatically give a metric, but a metric does not necessarily come from a norm. In Banach spaces, a metric is induced by the norm \[ d(x,y) = \|x - y\|. \]. This does not necessarily define the angles yet for Banach - inner product needed.

Key Point: Compactness and the unit ball in infinite-dimensional normed spaces

Compactness:
A subset \(K\) of a normed space \(V\) is compact if every sequence \((x_n) \subset K\) has a subsequence that converges to an element in \(K\) (sequential compactness). Equivalently, in topological terms, \(K\) is compact if **every open cover of \(K\) has a finite subcover**: for any collection of open sets \(\mathcal{U}\) with \(\displaystyle K \subseteq \bigcup_{U \in \mathcal{U}} U\), there exists a finite subcollection \(U_1, \dots, U_n \in \mathcal{U}\) such that \(\displaystyle K \subseteq U_1 \cup \dots \cup U_n\).

Important fact in infinite-dimensional spaces:
- Unlike finite-dimensional spaces, in infinite-dimensional normed spaces, a set being closed and bounded does not guarantee compactness.

Unit ball:
- The unit ballKey point: In infinite-dimensional spaces, the unit ball is never compact. - Intuition: You can always construct a sequence in the unit ball (e.g., basis vectors) with no convergent subsequence, so it fails the sequential compactness criterion.

Summary: - Compact = every sequence has a convergent subsequence within the set. - Closed + bounded ≠ compact in infinite dimensions. - Unit ball is a simple example of a bounded, closed set that is **never compact**.

Key Point: Connectedness and countability in infinite-dimensional normed spaces

Connectedness:
A topological space \(X\) is connected if it cannot be represented as the union of two nonempty disjoint open sets. Equivalently, there are no two nonempty open sets \(U, V \subset X\) such that \(X = U \cup V\) and \(U \cap V = \emptyset\).

In normed spaces (including infinite-dimensional ones), connectedness often comes from **path-connectedness**: any two points \(x, y \in X\) can be joined by a continuous path \(\gamma: [0,1] \to X\) with \(\gamma(0) = x, \gamma(1) = y\). - Infinite-dimensional normed spaces are typically **connected** and **path-connected**.

First-countable (1st ctb):
A topological space \(X\) is first-countable if each point \(x \in X\) has a **countable local base** of neighborhoods. - In normed spaces, the open balls \(B(x, 1/n)\), \(n = 1, 2, \dots\), form a countable local base, so **all normed spaces are first-countable**.

Second-countable (2nd ctb):
A space is second-countable if there exists a **countable base for the entire topology**. - Finite-dimensional normed spaces (like \(\mathbb{R}^n\)) are second-countable. - Infinite-dimensional normed spaces are generally **not second-countable**, because any base for the topology of open balls must be uncountable.

Summary:
- Connected = cannot split into two nonempty disjoint open sets. - Path-connected = any two points joined by a continuous path; implies connectedness. - 1st ctb = countable local base at each point → holds in all normed spaces. - 2nd ctb = countable base for the whole topology → fails in infinite-dimensional spaces.

Key Point: Product and Quotient Topologies

1. Product Topology:
Let \(\{X_i\}_{i \in I}\) be a family of topological spaces with topologies \(\tau_i\). The product space is \[ X = \prod_{i \in I} X_i = \{ (x_i)_{i \in I} : x_i \in X_i \}. \] The product topology on \(X\) is the coarsest (smallest) topology that makes all projection maps \(\pi_j : X \to X_j\), \(\pi_j((x_i)_{i \in I}) = x_j\), continuous.

  • Basis elements: finite intersections of sets of the form \(\pi_j^{-1}(U_j)\), where \(U_j \subset X_j\) is open.
  • Intuition: Open sets in the product correspond to “all but finitely many coordinates unrestricted, finitely many coordinates constrained.”

Example: \(\mathbb{R}^n = \prod_{i=1}^n \mathbb{R}\) with the usual topology is a product topology of \(n\) copies of \(\mathbb{R}\). Infinite products (e.g., sequences) use the same idea but with infinitely many coordinates.

2. Quotient Topology:
Let \(X\) be a topological space with topology \(\tau\), and let \(\sim\) be an equivalence relation on \(X\). The set of equivalence classes is \[ X / \sim = \{ [x] : x \in X \}. \] The quotient topology on \(X / \sim\) is defined as the finest topology that makes the canonical projection \(\pi: X \to X / \sim, \ \pi(x) = [x]\) continuous.

  • Open sets in \(X / \sim\) are exactly the subsets \(U \subset X / \sim\) for which \(\pi^{-1}(U)\) is open in \(X\).
  • Intuition: The quotient topology “inherits” openness from \(X\) via the projection map.

Example: Identify opposite edges of a square to form a torus. The quotient topology ensures that neighborhoods on the edges “glue together” properly.

12: Normed Vector Spaces

Question: Define a normed vector space and list the properties that the norm must satisfy.

Solution

Definition:
A normed vector space is a vector space \(X\) over \(\mathbb{R}\) or \(\mathbb{C}\) equipped with a function \(\|\cdot\| : X \to [0,\infty)\) called a norm that satisfies, for all \(x, y \in X\) and \(\alpha \in \mathbb{F}\):

  • Positive definiteness: \(\|x\| = 0 \iff x = 0\)
  • Homogeneity (absolute scalability): \(\|\alpha x\| = |\alpha| \cdot \|x\|\)
  • Triangle inequality: \(\|x + y\| \le \|x\| + \|y\|\)

The norm induces a metric \(d(x,y) = \|x - y\|\), so every normed vector space is also a metric space.

13: Equivalent Norms

(a): What does it mean for two norms to be equivalent?

(b): Show a proof that equivalent norms induce the same topology.

Solutions

(a) Equivalent Norms:
Two norms \(\|\cdot\|_1\) and \(\|\cdot\|_2\) on a vector space \(V\) are called equivalent if there exist positive constants \(c_1, c_2 > 0\) such that for all \(x \in V\): \[ c_1 \|x\|_1 \le \|x\|_2 \le c_2 \|x\|_1 \] Equivalent norms induce the same topology, meaning they have the same notions of convergence and open sets.

(b) Proof Idea:
Let \(x_n \to x\) in \(\|\cdot\|_1\). By equivalence, we have \[ \|x_n - x\|_2 \le c_2 \|x_n - x\|_1 \to 0, \] so \(x_n \to x\) in \(\|\cdot\|_2\) as well. Hence, convergence is preserved and the norms induce the same topology.

Finite-Dimensional Case:
If \(\dim V < \infty\), all norms on \(V\) are equivalent. For example, with a basis \(\{e_i\}\), we can define the infinity norm \(\|x\|_\infty = \max |x_i|\). Equivalence follows from finiteness of coordinates.

Infinite-Dimensional Case:
In infinite-dimensional spaces, equivalence may fail. For instance, compactness properties that hold in finite dimensions may fail in \(\ell^\infty\), so the theorem “all norms are equivalent” does not hold.

14: Continuity of Linear Operators

Question: Let \((V, \|\cdot\|_V)\) and \((W, \|\cdot\|_W)\) be normed vector spaces. Define what it means for a linear map \(T: V \to W\) to be bounded (continuous).

Solution

Definition:
A linear map \(T: V \to W\) is called bounded (or continuous) if there exists a constant \(M \ge 0\) such that for all \(x \in V\): \[ \|T(x)\|_W \le M \|x\|_V. \]

Remark: In normed vector spaces, linear operators are continuous if and only if they are bounded. This is a key result in functional analysis.

15: Equivalent Statements for Linear Operators

Question: Let \(T: V \to W\) be a linear map between normed vector spaces \((V, \|\cdot\|_V)\) and \((W, \|\cdot\|_W)\). Prove that the following statements are equivalent:

  1. \(T\) is bounded: there exists \(M \ge 0\) such that \(\|T(x)\|_W \le M \|x\|_V\) for all \(x \in V\).
  2. \(T\) is continuous at \(0 \in V\).
  3. \(T\) is continuous everywhere on \(V\).
Solution

Proof:

(1 ⇒ 3): Suppose \(T\) is bounded, so there exists \(M \ge 0\) with \(\|T(x)\|_W \le M \|x\|_V\). For any \(x_0 \in V\) and \(x \in V\):

\[ \|T(x) - T(x_0)\|_W = \|T(x - x_0)\|_W \le M \|x - x_0\|_V, \]

so \(T\) is continuous everywhere.

(3 ⇒ 2): Trivial, because continuity everywhere implies continuity at \(0\).

(2 ⇒ 1): Suppose \(T\) is continuous at \(0\). Then there exists \(\delta > 0\) such that \(\|x\|_V < \delta \implies \|T(x)\|_W < 1\). By linearity, for any non-zero \(x \in V\):

\[ T(x) = \frac{\delta}{\|x\|_V} T\Big(\frac{\|x\|_V}{\delta} x\Big) \implies \|T(x)\|_W \le \frac{\|x\|_V}{\delta}. \]

Hence \(T\) is bounded with \(M = 1/\delta\).

Conclusion: Boundedness, continuity at 0, and continuity everywhere are equivalent for linear maps between normed spaces.

16: Operator Norm

Question: Let \(T: V \to W\) be a linear map between normed vector spaces \((V, \|\cdot\|_V)\) and \((W, \|\cdot\|_W)\). Define the operator norm of \(T\) and prove the following:

  1. \(\|T\| = \sup_{\|x\|_V \le 1} \|T(x)\|_W = \sup_{x \neq 0} \frac{\|T(x)\|_W}{\|x\|_V}\).
  2. If \(\dim V < \infty\), then every linear map \(T: V \to W\) is continuous (bounded).
Solution

Definition:
The operator norm of \(T\) is defined by \[ \|T\| := \sup_{\|x\|_V \le 1} \|T(x)\|_W. \] Equivalently, for \(x \neq 0\): \[ \|T\| = \sup_{x \neq 0} \frac{\|T(x)\|_W}{\|x\|_V}. \] This gives the smallest constant \(M\) such that \(\|T(x)\|_W \le M \|x\|_V\) for all \(x \in V\).

Proof that \(\dim V < \infty \implies T\) is continuous:
1. Let \(\dim V = n < \infty\) and choose a basis \(\{e_1, \dots, e_n\}\) of \(V\).
2. With respect to this basis, \(T\) can be represented as a matrix \(A \in \mathbb{F}^{m \times n}\), mapping coordinates in \(V\) to coordinates in \(W\).
3. In finite dimensions, **all norms are equivalent**, so there exists constants \(c_1, c_2 > 0\) such that \[ c_1 \|x\|_{\text{coord}} \le \|x\|_V \le c_2 \|x\|_{\text{coord}}. \] 4. Then \[ \|T(x)\|_W \le \|A\|_{\text{matrix}} \|x\|_{\text{coord}} \le \|A\|_{\text{matrix}} c_2 \|x\|_V, \] giving a bound. Hence \(T\) is bounded (continuous).

Conclusion: In finite-dimensional spaces, every linear map is automatically continuous. The operator norm provides a quantitative measure of its “size.”

16: Closed Subspaces of Banach Spaces

Question: Let \(X\) be a Banach space and \(Y \subset X\) be a closed linear subspace. Prove that \(Y\) is also a Banach space.

Solution

Proof:

  1. Let \((y_n)\) be a Cauchy sequence in \(Y\). By definition of a Cauchy sequence: \[ \forall \epsilon > 0, \exists N \text{ s.t. } m,n \ge N \implies \|y_n - y_m\|_X < \epsilon. \]
  2. Since \(Y \subset X\), \((y_n)\) is also a Cauchy sequence in \(X\). Because \(X\) is Banach, there exists \(y \in X\) such that \[ y_n \to y \text{ in } X. \]
  3. Since \(Y\) is closed in \(X\), the limit of any convergent sequence in \(Y\) lies in \(Y\). Hence \[ y \in Y. \]
  4. Therefore, every Cauchy sequence in \(Y\) converges to a point in \(Y\). By definition, \(Y\) is Banach.

Conclusion: Any closed subspace of a Banach space is itself a Banach space.

17: Quotient Spaces of Banach Spaces

Question: Let \(X\) be a Banach space and \(Y \subset X\) a closed subspace. Define the quotient space \(X/Y\) with the norm \[ \|x + Y\| := \inf_{y \in Y} \|x + y\|_X. \] Prove that \(X/Y\) is a Banach space.

Solution

Proof:

  1. Let \((x_n + Y)\) be a Cauchy sequence in \(X/Y\). By definition of the quotient norm: \[ \forall \epsilon > 0, \exists N \text{ s.t. } m,n \ge N \implies \|x_n - x_m + Y\| < \epsilon. \]
  2. By the definition of the quotient norm, for each \(n\) we can pick representatives \(x_n' \in x_n + Y\) so that \(\|x_n' - x_m'\|_X < \epsilon + \frac{1}{n}\). Then \((x_n')\) is a Cauchy sequence in \(X\).
  3. Since \(X\) is Banach, \((x_n')\) converges to some \(x \in X\): \[ x_n' \to x \text{ in } X. \]
  4. The sequence of cosets \(x_n + Y\) converges to \(x + Y\) in the quotient norm. Indeed, \[ \|x_n + Y - (x + Y)\| = \inf_{y \in Y} \|x_n' - x + y\|_X \to 0, \] because \(Y\) is closed and contains all differences of representatives.
  5. Hence, every Cauchy sequence in \(X/Y\) converges in \(X/Y\), proving that \(X/Y\) is Banach.

Conclusion: The quotient of a Banach space by a closed subspace is again a Banach space.

18: Hahn–Banach Theorem

Question: State the Hahn–Banach theorem for real normed vector spaces. Then prove it (or provide a proof sketch) showing how a bounded linear functional defined on a subspace can be extended to the whole space without increasing its norm.

Solution

Statement (Real Case):
Let \(X\) be a real vector space, \(Y \subset X\) a linear subspace, and \(p: X \to \mathbb{R}\) a sublinear function (i.e., positively homogeneous and subadditive). If \(f: Y \to \mathbb{R}\) is a linear functional such that \[ f(y) \le p(y) \quad \forall y \in Y, \] then there exists a linear extension \(F: X \to \mathbb{R}\) of \(f\) (i.e., \(F|_Y = f\)) such that \[ F(x) \le p(x) \quad \forall x \in X. \]

Corollary for Normed Spaces:
If \(X\) is a normed space, \(Y \subset X\) a subspace, and \(f: Y \to \mathbb{R}\) (or \(\mathbb{C}\)) is linear and bounded, then there exists an extension \(F: X \to \mathbb{R}\) (or \(\mathbb{C}\)) with the same norm: \[ \|F\| = \|f\|. \]

Proof Sketch (Real Case):

  1. Start with a subspace \(Y \subset X\) and a linear functional \(f: Y \to \mathbb{R}\) with \(f(y) \le p(y)\).
  2. Pick \(x_0 \in X \setminus Y\). Define a candidate extension \(f_1\) on \(Y_1 = Y + \mathrm{span}\{x_0\}\) by \[ f_1(y + \alpha x_0) = f(y) + \alpha c \] for some \(c \in \mathbb{R}\) chosen to satisfy \[ f_1(y + \alpha x_0) \le p(y + \alpha x_0) \quad \forall y \in Y, \alpha \in \mathbb{R}. \]
  3. By sublinearity of \(p\), such a \(c\) exists, giving a one-dimensional extension.
  4. Use Zorn’s lemma: consider the set of all pairs \((Z, g)\) where \(Y \subset Z \subset X\) and \(g: Z \to \mathbb{R}\) is linear with \(g \le p\). This set is partially ordered by inclusion.
  5. Every chain has an upper bound (union of subspaces and consistent extensions), so by Zorn’s lemma there exists a maximal element \((M, F)\).
  6. Maximality implies \(M = X\); otherwise, we could extend \(F\) further by one dimension. Hence \(F\) is a linear extension of \(f\) to all of \(X\) satisfying \(F \le p\).

Conclusion: Any bounded linear functional on a subspace of a normed space can be extended to the whole space without increasing its norm. This is the Hahn–Banach theorem.

19: Open Mapping and Closed Graph Theorems

(a): Open Mapping Theorem
Let \(X\) and \(Y\) be Banach spaces and \(T: X \to Y\) be a bounded linear surjective operator. Prove that \(T\) is an open mapping, i.e., it maps open sets in \(X\) to open sets in \(Y\).

(b): Closed Graph Theorem
Let \(X\) and \(Y\) be Banach spaces and \(T: X \to Y\) a linear operator. Prove that if the graph \[ G(T) = \{(x, T(x)) \in X \times Y : x \in X\} \] is closed in \(X \times Y\), then \(T\) is bounded (continuous).

Solutions

(a) Open Mapping Theorem:

  1. Let \(B_X(0,1) = \{x \in X : \|x\|_X < 1\}\) be the open unit ball in \(X\).
  2. Since \(T\) is surjective, \(T(B_X(0,1))\) is a subset of \(Y\) whose linear span is \(Y\).
  3. By the Baire category theorem, the closure of \(T(B_X(0,1))\) has nonempty interior in \(Y\).
  4. Scaling argument: If \(y_0 \in \text{int}(\overline{T(B_X(0,1))})\), then for any \(y \in Y\), \[ y = \|y\|_Y \frac{y}{\|y\|_Y} \in \|y\|_Y \, T(B_X(0,1)), \] showing \(T\) maps open sets in \(X\) to open sets in \(Y\).
  5. Conclusion: \(T\) is an open mapping.

(b) Closed Graph Theorem:

  1. Consider the graph \(G(T) \subset X \times Y\). The product space \(X \times Y\) is Banach under the norm \(\|(x,y)\| = \|x\|_X + \|y\|_Y\).
  2. Define the projection \(\pi_X: G(T) \to X\) by \(\pi_X(x, T(x)) = x\). This is linear and bijective from \(G(T)\) onto \(X\).
  3. By the open mapping theorem, \(\pi_X^{-1}: X \to G(T)\) is bounded, because \(G(T)\) is Banach (closed in \(X \times Y\)) and \(\pi_X\) is linear and bijective.
  4. But \(\pi_X^{-1}(x) = (x, T(x))\), so boundedness of \(\pi_X^{-1}\) implies \[ \|(x, T(x))\| = \|x\|_X + \|T(x)\|_Y \le M \|x\|_X \implies \|T(x)\|_Y \le (M-1)\|x\|_X. \]
  5. Hence \(T\) is bounded (continuous).

20: Uniform Boundedness Principle (Banach–Steinhaus Theorem)

Question: Let \(X\) be a Banach space, \(Y\) a normed space, and \(\{T_\alpha\}_{\alpha \in A}\) a family of bounded linear operators from \(X\) to \(Y\). Suppose that for every \(x \in X\), \[ \sup_{\alpha \in A} \|T_\alpha(x)\|_Y < \infty. \] Prove that the operators are uniformly bounded, i.e., \[ \sup_{\alpha \in A} \|T_\alpha\| < \infty. \]

Solution

Proof (Baire Category argument):

  1. Define the sets \[ E_n := \{ x \in X : \|T_\alpha(x)\|_Y \le n \text{ for all } \alpha \in A \}, \quad n \in \mathbb{N}. \]
  2. By assumption, for each \(x \in X\) there exists \(n\) such that \(x \in E_n\). Hence \[ X = \bigcup_{n=1}^\infty E_n. \]
  3. Each \(E_n\) is **closed**: if \(x_k \in E_n\) and \(x_k \to x\), then by continuity of each \(T_\alpha\), \[ \|T_\alpha(x)\|_Y = \lim_{k \to \infty} \|T_\alpha(x_k)\|_Y \le n. \]
  4. Since \(X\) is Banach, by the **Baire category theorem**, there exists \(N\) such that \(E_N\) has **nonempty interior**. That is, there exists a ball \(B_X(x_0, r) \subset E_N\).
  5. For any \(x \in X\) and any \(\alpha \in A\), write \[ x = x_0 + (x - x_0) \in B_X(x_0, r) + B_X(0, \|x-x_0\|), \] and use linearity and the triangle inequality to get a bound \[ \|T_\alpha(x)\|_Y \le \|T_\alpha(x_0)\|_Y + \|T_\alpha(x-x_0)\|_Y \le N + \frac{N}{r} \|x-x_0\|_X. \]
  6. Hence there exists a constant \(M > 0\) independent of \(\alpha\) such that \[ \|T_\alpha\| \le M \quad \forall \alpha \in A. \]

Conclusion: The family \(\{T_\alpha\}_{\alpha \in A}\) is uniformly bounded. This is the Uniform Boundedness Principle.

The continuous dual space of a normed space \(X\) is \[ X^* := \{ f: X \to \mathbb{F} \text{ linear and continuous (bounded)} \}, \] where \(\mathbb{F} = \mathbb{R}\) or \(\mathbb{C}\). By the equivalence theorem for linear operators on normed spaces, a linear functional \(f\) is continuous if and only if it is bounded, i.e., there exists \(M \ge 0\) such that \[ |f(x)| \le M \|x\| \quad \forall x \in X. \]

For \(f \in X^*\), its norm is defined as \[ \|f\| := \sup_{x \neq 0} \frac{|f(x)|}{\|x\|}. \] Equivalently, using homogeneity of linear maps: \[ \|f\| = \sup_{\|x\| \le 1} |f(x)|. \] This is exactly the operator norm of the linear functional \(f\). 23: The Continuous Dual of a Normed Space is Banach

Question: Let \(X\) be a normed space (not necessarily complete). Show that its continuous dual \(X^*\), equipped with the operator norm, is a Banach space. In particular, if \((f_n) \subset X^*\) is a Cauchy sequence, prove that it converges to some \(f \in X^*\).

Solution

Proposition: Let \(X\) be a normed space. Then \((X^*, \|\cdot\|)\) is a Banach space, i.e., every Cauchy sequence in \(X^*\) converges to a bounded linear functional in \(X^*\).

Proof:

  1. Let \((f_n) \subset X^*\) be a Cauchy sequence in the operator norm. That is, \[ \forall \epsilon > 0, \exists N \text{ s.t. } m,n \ge N \implies \|f_n - f_m\| = \sup_{\|x\| \le 1} |f_n(x) - f_m(x)| < \epsilon. \]
  2. Fix \(x \in X\). Then for all \(m,n \ge N\), \[ |f_n(x) - f_m(x)| \le \|f_n - f_m\| \cdot \|x\| < \epsilon \|x\|. \] Hence, for each \(x \in X\), the sequence \((f_n(x))\) is Cauchy in \(\mathbb{F} = \mathbb{R}\) or \(\mathbb{C}\), which is complete. So we can define \[ f(x) := \lim_{n \to \infty} f_n(x). \]
  3. Linearity of \(f\) follows from linearity of each \(f_n\) and the limit: \[ f(\alpha x + \beta y) = \lim_{n \to \infty} f_n(\alpha x + \beta y) = \alpha \lim f_n(x) + \beta \lim f_n(y) = \alpha f(x) + \beta f(y). \]
  4. \(f\) is bounded: For all \(x \in X\), \[ |f(x)| = \lim_{n \to \infty} |f_n(x)| \le \limsup_{n \to \infty} \|f_n\| \cdot \|x\|. \] Since \((f_n)\) is Cauchy, \(\sup_n \|f_n\| < \infty\). Hence \(\|f\| < \infty\) and \(f \in X^*\).
  5. Finally, convergence in the operator norm: \[ \|f_n - f\| = \sup_{\|x\| \le 1} |f_n(x) - f(x)| \to 0 \quad \text{as } n \to \infty. \]

Conclusion: Every Cauchy sequence in \(X^*\) converges in \(X^*\). Therefore, \((X^*, \|\cdot\|)\) is a Banach space, even if \(X\) is not complete.

24: Canonical Embedding into the Bidual

Question: Let \(X\) be a normed space and \(X^*\) its continuous dual. Define the canonical embedding \(J: X \to X^{**}\) (the bidual). Prove rigorously that \(J\) is linear and isometric.

Solution

Proposition: For a normed space \(X\), the map \[ J: X \to X^{**}, \quad J(x)(f) := f(x) \text{ for all } f \in X^* \] is linear and satisfies \(\|J(x)\| = \|x\|\). That is, \(J\) is an isometric embedding of \(X\) into its bidual.

Proof:

  1. Linearity: For \(x, y \in X\) and \(\alpha, \beta \in \mathbb{F}\), \[ J(\alpha x + \beta y)(f) = f(\alpha x + \beta y) = \alpha f(x) + \beta f(y) = \alpha J(x)(f) + \beta J(y)(f), \] so \(J(\alpha x + \beta y) = \alpha J(x) + \beta J(y)\).
  2. Norm-preserving (isometry): By definition of the operator norm, \[ \|J(x)\| = \sup_{\|f\| \le 1} |J(x)(f)| = \sup_{\|f\| \le 1} |f(x)|. \] But by the definition of the dual norm, \[ |f(x)| \le \|f\| \|x\| \le \|x\| \quad \forall \|f\| \le 1, \] so \(\|J(x)\| \le \|x\|\).
  3. To see equality, use the Hahn–Banach theorem: there exists \(f_0 \in X^*\) with \(\|f_0\| = 1\) such that \[ f_0(x) = \|x\|. \] Then \[ \|J(x)\| \ge |J(x)(f_0)| = |f_0(x)| = \|x\|. \]

Conclusion: \(\|J(x)\| = \|x\|\), so \(J\) is linear and isometric. This provides a natural way to identify \(X\) with a subspace of \(X^{**}\).

25: Weak and Weak-* Topologies

Question: Let \(X\) be a normed space and \(X^*\) its continuous dual. Consider the following:

  1. Weak topology: Define the weak topology on \(X\) and describe its basic open sets.
  2. Weak-* topology: Define the weak-* topology on \(X^*\) and describe its basic open sets.
  3. Infinite-dimensional case: Explain why the weak and weak-* topologies are strictly weaker than the norm topology if \(X\) is infinite-dimensional, and give an intuitive reason.
Solution

(1) Weak Topology on \(X\):

The weak topology on \(X\) is the coarsest topology making all functionals \(f \in X^*\) continuous. A net \((x_\alpha) \subset X\) converges weakly to \(x \in X\), written \(x_\alpha \rightharpoonup x\), if \[ f(x_\alpha) \to f(x) \quad \forall f \in X^*. \] Basic open sets are of the form \[ \{ y \in X : |f_i(y) - f_i(x_0)| < \epsilon, \ i=1,\dots,n \}, \] for finitely many functionals \(f_1,\dots,f_n\), \(\epsilon > 0\), and \(x_0 \in X\).

(2) Weak-* Topology on \(X^*\):

The weak-* topology on \(X^*\) is the coarsest topology making the evaluation maps \(f \mapsto f(x)\) continuous for all \(x \in X\). A net \((f_\alpha) \subset X^*\) converges weak-* to \(f \in X^*\), written \(f_\alpha \overset{*}{\rightharpoonup} f\), if \[ f_\alpha(x) \to f(x) \quad \forall x \in X. \] Basic open sets are of the form \[ \{ g \in X^* : |g(x_i) - f(x_i)| < \epsilon, \ i=1,\dots,n \}, \] for finitely many points \(x_1,\dots,x_n \in X\), \(\epsilon > 0\), and \(f \in X^*\).

(3) Infinite-dimensional case:

  • In infinite-dimensional spaces, weak convergence is strictly weaker than norm convergence: a net can converge weakly without converging in norm. Intuition: only finitely many functionals “test” convergence at a time.
  • Similarly, weak-* convergence is weaker than norm convergence on \(X^*\): a sequence may converge pointwise on \(X\) but not in operator norm.
  • This explains phenomena like the Banach–Alaoglu theorem: closed balls in \(X^*\) are compact in the weak-* topology, even though they are never compact in norm if \(X\) is infinite-dimensional.

Conclusion: Weak and weak-* topologies provide weaker notions of convergence than norm convergence, making them crucial in infinite-dimensional functional analysis.

26: Weak Convergence of Sequences

Question: Let \(X\) be a normed space. Define what it means for a sequence \((x_n) \subset X\) to converge weakly to \(x \in X\). Give an example in an infinite-dimensional space where weak convergence occurs but norm convergence does not.

Solution

Definition (Weak Convergence):

A sequence \((x_n) \subset X\) converges weakly to \(x \in X\), written \(x_n \rightharpoonup x\), if \[ f(x_n) \to f(x) \quad \text{for all } f \in X^*, \] i.e., it converges pointwise when tested against every continuous linear functional.

Example in \(l^2\):

  • Let \(e_n = (0,0,\dots,1,0,\dots)\) be the standard basis of \(l^2\) (1 in the \(n\)-th coordinate, 0 elsewhere).
  • Define the sequence \(x_n = e_n\). Then \(\|x_n\| = 1\) for all \(n\), so \(x_n\) does **not** converge in norm.
  • However, for any \(f \in (l^2)^*\) (by Riesz representation, \(f(y) = \langle y, z\rangle\) for some \(z \in l^2\)), we have \[ f(e_n) = \langle e_n, z \rangle = z_n \to 0 \quad \text{as } n \to \infty, \] since \(z \in l^2\) implies \(z_n \to 0\). Therefore, \(e_n \rightharpoonup 0\) weakly.

Conclusion: This shows a sequence can converge weakly without converging in norm, a phenomenon that occurs only in infinite-dimensional spaces.

27: Banach–Alaoglu Theorem (proof)

Question: State the Banach–Alaoglu theorem and prove that the closed unit ball in the dual space \(X^*\) is compact in the weak-* topology.

Solution

Theorem (Banach–Alaoglu):

Let \(X\) be a normed space. Then the closed unit ball in the dual space \[ B_{X^*} = \{ f \in X^* : \|f\| \le 1 \} \] is compact in the weak-* topology.

Proof (Sketch / Rigorous Outline):

  1. Consider the product space \(\Pi_{x \in X} \overline{\mathbb{D}}\), where \(\overline{\mathbb{D}} = \{ z \in \mathbb{F} : |z| \le 1 \}\) and \(\mathbb{F} = \mathbb{R} \text{ or } \mathbb{C}\).
  2. By Tychonoff’s theorem, the product of compact spaces is compact in the product topology. So \[ \Pi_{x \in X} \overline{\mathbb{D}} \] is compact.
  3. Each \(f \in B_{X^*}\) defines a point in this product via evaluation: \[ f \mapsto (f(x))_{x \in X} \in \Pi_{x \in X} \overline{\mathbb{D}}. \]
  4. The weak-* topology on \(B_{X^*}\) is exactly the subspace topology induced by this product (since weak-* convergence is pointwise convergence on \(X\)).
  5. Thus, \(B_{X^*}\) is a closed subset of a compact space (the image under the embedding), so it is compact in the weak-* topology.

Remark / Intuition: Weak-* convergence tests only finitely many points of \(X\) at a time. Tychonoff’s theorem guarantees compactness in the infinite-dimensional setting, even though the unit ball is never norm-compact in infinite dimensions.

28: Banach–Alaoglu Theorem (Extended)

Question: State the Banach–Alaoglu theorem and explain its consequences in the following contexts:

  1. Separability of \(X\) and weak-* metrisability of the dual unit ball.
  2. Sequential compactness in weak-* topology for separable spaces.
Solution

Banach–Alaoglu Theorem: For a normed space \(X\), the closed unit ball in the dual \[ B_{X^*} = \{ f \in X^* : \|f\| \le 1 \} \] is compact in the weak-* topology.

(1) Separability and weak-* metrisability:

  • If \(X\) is **separable**, then there exists a **countable dense subset** \(\{x_n\} \subset X\).
  • The weak-* topology on \(B_{X^*}\) can then be defined using only evaluations at \(\{x_n\}\), giving a **countable subbasis**.
  • By general topology, a compact space with a countable subbasis is **metrisable**. So the weak-* topology on \(B_{X^*}\) is **metrisable** for separable \(X\).

(2) Sequential compactness:

  • metrisable compact spaces are **sequentially compact**. Therefore, in a separable normed space \(X\), any sequence \((f_n) \subset B_{X^*}\) has a **weak-* convergent subsequence**.
  • Intuition: Although \(B_{X^*}\) may not be norm-compact, weak-* convergence only tests finitely many points at a time, making sequential limits possible in separable spaces.

Summary / Intuition:

  • Banach–Alaoglu gives **compactness of the dual unit ball** in weak-* topology.
  • If \(X\) is separable, the weak-* topology becomes **metrisable**, so compactness implies **sequential compactness**.
  • This is a powerful tool: it guarantees existence of weak-* convergent subsequences in separable dual spaces.

Key Points: Axiom of Choice & Tychonoff’s Theorem

(Quiz-style): State the key points about the Axiom of Choice and Tychonoff’s theorem, and briefly explain why they are considered equivalent in mathematics.

Solution / Key Points
  1. Axiom of Choice (AC): Given any collection of non-empty sets \(\{X_i\}_{i \in I}\), there exists a function \(f\) (a choice function) such that \(f(i) \in X_i\) for each \(i \in I\). Intuition: You can simultaneously pick one element from each set, even infinitely many sets.
  2. Tychonoff’s Theorem: The product of any collection of compact topological spaces \(\prod_{i \in I} X_i\) is compact in the product topology. Intuition: “Infinite products of compact sets remain compact.”
  3. Equivalence: Proving Tychonoff’s theorem for arbitrary (possibly infinite) products requires the ability to **choose an element from each factor**. Hence, Tychonoff’s theorem in full generality **is equivalent to the Axiom of Choice**. Both are deeply connected to making infinitely many selections consistently.

30: Filters and Ultrafilters

Question: Answer the following:

  1. Filter: Define what a filter is on a set \(X\) and give a brief intuition.
  2. Ultrafilter: Define an ultrafilter and explain how it differs from a general filter.
Solution / Key Points
  1. Filter on a set \(X\): A filter \(\mathcal{F}\) is a collection of subsets of \(X\) satisfying:
    • \(X \in \mathcal{F}\), \(\emptyset \notin \mathcal{F}\)
    • Closed under finite intersections: If \(A, B \in \mathcal{F}\), then \(A \cap B \in \mathcal{F}\)
    • Upward closed: If \(A \in \mathcal{F}\) and \(A \subset B \subset X\), then \(B \in \mathcal{F}\)
    Intuition: A filter collects the “large” or “significant” subsets of \(X\) — sets that matter for convergence.
  2. Ultrafilter on a set \(X\): An ultrafilter \(\mathcal{U}\) is a **maximal filter**, meaning:
    • It is a filter.
    • For every subset \(A \subset X\), either \(A \in \mathcal{U}\) or \(X \setminus A \in \mathcal{U}\) (never both).
    Difference from a general filter: An ultrafilter makes a “decision” for every subset: it is either “large” (in the ultrafilter) or “small” (its complement is in the ultrafilter). Intuition: An ultrafilter is a filter that is as “decisive” as possible, which allows it to define limits in arbitrary product spaces.

Tychonoff’s Theorem: Infinite Product Case

Question: Prove that the product of an arbitrary collection of compact spaces is compact (infinite case), and explain why the Axiom of Choice is needed.

Solution / Proof

Theorem: Let \(\{X_i\}_{i \in I}\) be compact topological spaces (possibly infinitely many). Then the product space \[ X = \prod_{i \in I} X_i \] with the product topology is compact.

Proof using ultrafilters:

  1. Let \(\mathcal{U}\) be an **ultrafilter** on \(X\). To show compactness, it suffices to show that \(\mathcal{U}\) converges to some point \(x \in X\).
  2. For each coordinate \(i \in I\), define the **projected ultrafilter** on \(X_i\): \[ \mathcal{U}_i = \{ A \subset X_i : \pi_i^{-1}(A) \in \mathcal{U} \}. \]
  3. Since each \(X_i\) is compact, the ultrafilter \(\mathcal{U}_i\) **converges** to some point \(x_i \in X_i\).
  4. Define the point in the product: \[ x = (x_i)_{i \in I} \in \prod_{i \in I} X_i. \]
  5. Check convergence: - A basic open neighborhood of \(x\) in the product topology depends only on finitely many coordinates. - For each finite subset \(F \subset I\), the sets \(\pi_i^{-1}(U_i) \in \mathcal{U}\) (with \(U_i\) neighborhoods of \(x_i\)) intersect to give a set in \(\mathcal{U}\). - This shows that \(\mathcal{U}\) contains every basic neighborhood of \(x\), so \(\mathcal{U} \to x\).
  6. Since **every ultrafilter converges**, the product space \(X\) is compact.

Why Axiom of Choice is needed:

  • Constructing the point \(x = (x_i)\) requires **choosing a limit \(x_i\) in each factor \(X_i\)** simultaneously.
  • For infinitely many indices, there is no canonical rule for these choices, so we need a **choice function**, which is guaranteed by AC.
  • Thus, Tychonoff’s theorem for infinite products is **equivalent to the Axiom of Choice**.

Intuition: Each factor is compact → each coordinate has a “limit.” Infinite product → pick a point in each coordinate consistently → use ultrafilters + AC → product is compact.

31: Goldstine's Theorem

Question: Let \(X\) be a normed space with unit ball \(B_X = \{x \in X : \|x\| \le 1\}\) and let \(J : X \to X^{**}\) be the canonical embedding into the bidual. State and prove Goldstine’s theorem, including the case when \(X\) is not reflexive.

Solution / Proof

Goldstine’s Theorem:

  • Let \(X\) be a normed space, \(B_X\) its closed unit ball, and \(J: X \to X^{**}\) the canonical embedding. Then \[ J(B_X) \text{ is weak-* dense in } B_{X^{**}}. \]
  • Remark: Even if \(X\) is **not reflexive** (so \(X \subsetneq X^{**}\)), its unit ball is still weak-* dense in the bidual unit ball. This means we can approximate any element of \(B_{X^{**}}\) arbitrarily closely in the weak-* topology using elements from \(X\).

Proof (Sketch using Hahn–Banach):

  1. Let \(F \in B_{X^{**}}\) and consider a weak-* neighborhood of \(F\) of the form \[ U = \{ G \in X^{**} : |G(f_i) - F(f_i)| < \epsilon, \, i = 1,\dots,n \} \] for finitely many \(f_1,\dots,f_n \in X^*\) and \(\epsilon > 0\).
  2. Define the linear map \(T: \text{span}\{f_1,\dots,f_n\} \subset X^* \to \mathbb{F}\) by \[ T\Big(\sum_{i=1}^n \alpha_i f_i\Big) = \sum_{i=1}^n \alpha_i F(f_i). \]
  3. \(T\) is bounded by the norm of \(F\), so by the **Hahn–Banach theorem**, \(T\) can be extended to a functional \(\tilde{T} \in X^{**}\) with \(\|\tilde{T}\| \le \|F\| \le 1\).
  4. By the **definition of the canonical embedding** \(J\), there exists \(x \in B_X\) such that \[ J(x)(f_i) = f_i(x) \approx F(f_i), \quad i = 1,\dots,n. \]
  5. Since \(U\) was arbitrary, this shows that for every weak-* neighborhood \(U\) of \(F\), there exists \(x \in B_X\) with \(J(x) \in U\). Hence \(J(B_X)\) is **weak-* dense** in \(B_{X^{**}}\).

Intuition: The Hahn–Banach theorem allows us to “approximate” the action of any element in the bidual by evaluating on finitely many functionals, which is exactly what the weak-* topology tests. Even in non-reflexive spaces, the original unit ball is dense enough to “cover” all weak-* neighborhoods.

Reflexive Banach Spaces

Question: Let \(X\) be a Banach space. State the equivalent characterisations of reflexivity and prove that they are indeed equivalent.

Solution / Key Points

Definition: A Banach space \(X\) is reflexive if the canonical embedding \[ J: X \to X^{**}, \quad J(x)(f) = f(x), \quad f \in X^*, \] is **surjective**, i.e., \(J(X) = X^{**}\).

Equivalent statements:

  1. Reflexivity: \(X \cong X^{**}\) via the canonical embedding.
  2. Unit ball weakly compact: The closed unit ball \(B_X = \{x \in X : \|x\| \le 1\}\) is compact in the **weak topology**.
  3. Dual reflexivity: \(X^*\) is reflexive (i.e., \(X^* \cong X^{***}\)).

Proof of equivalence:

  1. (1 ⇒ 2) Reflexive ⇒ unit ball weakly compact: - If \(X\) is reflexive, \(B_X\) is mapped onto \(B_{X^{**}}\) via \(J\). - By **Banach–Alaoglu**, \(B_{X^{**}}\) is weak-* compact. - The weak topology on \(X\) coincides with the restriction of the weak-* topology on \(X^{**}\) via \(J\), so \(B_X\) is weakly compact.
  2. (2 ⇒ 1) Unit ball weakly compact ⇒ reflexive: - Assume \(B_X\) is weakly compact. - By Goldstine, \(J(B_X)\) is weak-* dense in \(B_{X^{**}}\). - Since \(B_X\) is weakly compact, it is closed in the weak-* topology. - Density + closedness ⇒ \(J(B_X) = B_{X^{**}}\). - Scaling shows \(J(X) = X^{**}\), so \(X\) is reflexive.
  3. (1 ⇔ 3) Reflexivity ⇔ dual reflexivity: - If \(X\) is reflexive, \(X \cong X^{**}\). - Then \(X^* \cong X^{***}\), so \(X^*\) is reflexive. - Conversely, if \(X^*\) is reflexive, \(X \cong X^{**}\) by duality.

Conclusion: These statements are equivalent, providing multiple perspectives on reflexivity: via the canonical embedding, via weak compactness of the unit ball, or via dual reflexivity.

32: Hilbert Spaces and Inner Product

Question: Define a Hilbert space. List the properties of an inner product and show how it induces a norm.

Solution / Key Points

Definition: A Hilbert space is a complete inner product space \(H\) over \(\mathbb{R}\) or \(\mathbb{C}\). That is, \(H\) is a vector space with an inner product \(\langle \cdot , \cdot \rangle\) such that the metric induced by the inner product is complete.

Inner product properties: For all \(x, y, z \in H\) and \(\alpha \in \mathbb{F} (\mathbb{R} \text{ or } \mathbb{C})\):

  1. Conjugate symmetry / symmetry: \[ \langle x, y \rangle = \overline{\langle y, x \rangle} \quad (\text{real case: } \langle x, y \rangle = \langle y, x \rangle) \]
  2. Linearity in the first argument: \[ \langle \alpha x + y, z \rangle = \alpha \langle x, z \rangle + \langle y, z \rangle \]
  3. Positive definiteness: \[ \langle x, x \rangle \ge 0, \quad \langle x, x \rangle = 0 \iff x = 0 \]

Norm induced by inner product: \[ \|x\| = \sqrt{\langle x, x \rangle}, \quad x \in H \] This norm satisfies all properties of a norm (positivity, homogeneity, triangle inequality via Cauchy–Schwarz inequality), making \(H\) a normed space. Completeness with respect to this norm makes it a Hilbert space.

33: Cauchy–Schwarz Inequality

Question: Let \(H\) be a Hilbert space over \(\mathbb{R}\) or \(\mathbb{C}\) with inner product \(\langle \cdot, \cdot \rangle\). Prove the Cauchy–Schwarz inequality in both finite and infinite-dimensional cases.

Solution / Proof

Theorem (Cauchy–Schwarz inequality): For all \(x, y \in H\), \[ |\langle x, y \rangle| \le \|x\| \, \|y\|. \] Equality holds iff \(x\) and \(y\) are linearly dependent.

Proof (Finite-dimensional case)

  1. If \(y = 0\), inequality is trivial. Assume \(y \neq 0\).
  2. Consider the scalar \(\alpha = \frac{\langle x, y \rangle}{\|y\|^2}\) and the vector \(z = x - \alpha y\).
  3. By positive definiteness, \(\|z\|^2 \ge 0\), so \[ \|x - \alpha y\|^2 = \langle x - \alpha y, x - \alpha y \rangle \ge 0. \]
  4. Expand: \[ \|x\|^2 - \frac{|\langle x, y \rangle|^2}{\|y\|^2} \ge 0 \implies |\langle x, y \rangle|^2 \le \|x\|^2 \|y\|^2. \]
  5. Take square roots to get \(|\langle x, y \rangle| \le \|x\| \, \|y\|.\)

Proof (Infinite-dimensional case)

  1. Let \(\{e_i\}_{i \in I}\) be an orthonormal basis for \(H\) (or consider an orthonormal sequence spanning the subspace containing \(x, y\)).
  2. Expand \(x = \sum_{i} x_i e_i\), \(y = \sum_{i} y_i e_i\), where the sums converge in norm (since \(H\) is complete).
  3. Compute \[ \langle x, y \rangle = \sum_i \overline{x_i} y_i. \]
  4. Apply the finite-dimensional Cauchy–Schwarz inequality to the finite partial sums: \[ \Big|\sum_{i=1}^n \overline{x_i} y_i\Big| \le \Big(\sum_{i=1}^n |x_i|^2\Big)^{1/2} \Big(\sum_{i=1}^n |y_i|^2\Big)^{1/2}. \]
  5. Take the limit \(n \to \infty\) using completeness and norm convergence: \[ |\langle x, y \rangle| \le \|x\| \, \|y\|. \]

Conclusion: The Cauchy–Schwarz inequality holds in both finite and infinite-dimensional Hilbert spaces. It is a cornerstone of Hilbert space theory, underlying the triangle inequality, orthogonality, and projections.

34: Bessel's Inequality

Question: Let \(H\) be a Hilbert space over \(\mathbb{R}\) or \(\mathbb{C}\) with inner product \(\langle \cdot, \cdot \rangle\). Let \(\{e_i\}_{i \in I}\) be an orthonormal sequence in \(H\). Prove Bessel's inequality for any \(x \in H\).

Solution / Proof

Theorem (Bessel's inequality):

For any \(x \in H\) and orthonormal sequence \(\{e_i\}_{i \in I}\), \[ \sum_{i \in I} |\langle x, e_i \rangle|^2 \le \|x\|^2. \]

Proof:

  1. Let \(\{e_1, \dots, e_n\}\) be a finite subset of the orthonormal sequence. Consider the finite linear combination (the orthogonal projection of \(x\) onto \(\text{span}\{e_1, \dots, e_n\}\)): \[ P_n x = \sum_{i=1}^n \langle x, e_i \rangle e_i. \]
  2. By Pythagoras’ theorem (since \(x - P_n x\) is orthogonal to each \(e_i\)), we have \[ \|x\|^2 = \|P_n x\|^2 + \|x - P_n x\|^2 \ge \|P_n x\|^2. \]
  3. Compute the norm of the projection: \[ \|P_n x\|^2 = \left\| \sum_{i=1}^n \langle x, e_i \rangle e_i \right\|^2 = \sum_{i=1}^n |\langle x, e_i \rangle|^2. \]
  4. Thus, for any finite subset, \[ \sum_{i=1}^n |\langle x, e_i \rangle|^2 \le \|x\|^2. \]
  5. For an infinite orthonormal sequence, take the supremum over all finite subsets: \[ \sum_{i \in I} |\langle x, e_i \rangle|^2 = \sup_{n} \sum_{i=1}^n |\langle x, e_i \rangle|^2 \le \|x\|^2. \]

Conclusion: Bessel’s inequality shows that the “energy” of \(x\) along any orthonormal sequence is bounded by the total squared norm of \(x\). It is the key step toward Parseval’s identity and the theory of orthonormal bases in Hilbert spaces.

35: Parseval's Identity and Orthonormal Bases

Question: Let \(H\) be a Hilbert space.

  1. State and prove that every Hilbert space admits an orthonormal basis.
  2. Let \(\{e_i\}_{i \in I}\) be an orthonormal basis of \(H\). Prove Parseval’s identity for any \(x \in H\).
Solution / Proof

Part 1: Existence of an Orthonormal Basis

Theorem: Every Hilbert space \(H\) admits an orthonormal basis \(\{e_i\}_{i \in I}\), i.e., a maximal orthonormal set whose span is dense in \(H\).

Proof:

  1. Start with any non-zero vector \(x_1 \in H\) and normalise it: \(e_1 = x_1 / \|x_1\|\).
  2. Use Zorn's lemma (or transfinite induction) to extend any orthonormal set to a **maximal orthonormal set** \(\{e_i\}_{i \in I}\). This is possible because the union of a chain of orthonormal sets is orthonormal.
  3. Maximality implies that the span of \(\{e_i\}\) is dense in \(H\); otherwise, there exists \(x \notin \overline{\text{span}\{e_i\}}\), and we could normalise its orthogonal component to extend the set, contradicting maximality.
  4. Thus \(\{e_i\}_{i \in I}\) is an orthonormal basis of \(H\).

Remark: This works even for infinite-dimensional Hilbert spaces. Zorn's lemma ensures the existence of maximal orthonormal sets.

Part 2: Parseval's Identity

Theorem (Parseval's identity):

Let \(\{e_i\}_{i \in I}\) be an orthonormal basis of \(H\). Then for any \(x \in H\), \[ x = \sum_{i \in I} \langle x, e_i \rangle e_i \quad \text{(convergent in norm),} \] and \[ \|x\|^2 = \sum_{i \in I} |\langle x, e_i \rangle|^2. \]

Proof:

  1. Step 1: Bessel’s inequality: For any finite subset \(\{e_1, \dots, e_n\}\), \[ \sum_{i=1}^n |\langle x, e_i \rangle|^2 \le \|x\|^2. \]
  2. Step 2: Partial sums: Let \(P_n x = \sum_{i=1}^n \langle x, e_i \rangle e_i\). Then \(x - P_n x\) is orthogonal to each \(e_i\) in the span.
  3. Step 3: Completeness: Since \(\{e_i\}_{i \in I}\) is an orthonormal basis, the span is dense. Hence \(\|x - P_n x\| \to 0\) as \(n \to \infty\), giving \[ x = \sum_{i \in I} \langle x, e_i \rangle e_i. \]
  4. Step 4: Norm equality: Using Pythagoras for orthogonal components: \[ \|x\|^2 = \|P_n x\|^2 + \|x - P_n x\|^2 \to \sum_{i \in I} |\langle x, e_i \rangle|^2. \]

Conclusion: Every Hilbert space admits an orthonormal basis, and Parseval’s identity provides a complete “energy decomposition” of vectors along that basis.

36: Projection Theorem in Hilbert Spaces

Question: Let \(H\) be a Hilbert space and \(M \subset H\) a closed subspace. Prove that for every \(x \in H\), there exists a unique decomposition \[ x = y + z, \quad y \in M, \, z \in M^\perp, \] where \(M^\perp = \{ w \in H : \langle w, m \rangle = 0 \ \forall m \in M\}\). Show that \(y\) is the orthogonal projection of \(x\) onto \(M\).

Solution / Proof

Projection Theorem: For every closed subspace \(M \subset H\) and \(x \in H\), there exists a **unique** decomposition \[ x = y + z, \quad y \in M, \ z \in M^\perp, \] where \(y\) is the orthogonal projection of \(x\) onto \(M\).

Proof:

  1. Step 1: Existence of closest point: Consider the distance from \(x\) to \(M\): \[ d = \inf_{m \in M} \|x - m\|. \] Since \(M\) is closed and \(H\) is complete, there exists \(y \in M\) such that \[ \|x - y\| = d. \]
  2. Step 2: characterisation via inner product: For any \(m \in M\) and scalar \(\alpha\), consider the function \[ f(\alpha) = \| x - (y + \alpha m) \|^2 \ge \|x - y\|^2. \] Expand: \[ f(\alpha) = \|x - y\|^2 - 2 \text{Re}(\alpha \langle x - y, m \rangle) + |\alpha|^2 \|m\|^2. \] Minimality at \(\alpha = 0\) implies \[ \text{Re}(\langle x - y, m \rangle) = 0 \quad \forall m \in M. \]
  3. Step 3: Orthogonality: Replacing \(m\) by \(i m\) in the complex case, we also get \(\text{Im}(\langle x - y, m \rangle) = 0\). Hence \(\langle x - y, m \rangle = 0\) for all \(m \in M\), i.e., \[ z := x - y \in M^\perp. \]
  4. Step 4: Uniqueness: Suppose \(x = y_1 + z_1 = y_2 + z_2\) with \(y_i \in M\), \(z_i \in M^\perp\). Then \[ y_1 - y_2 = z_2 - z_1 \in M \cap M^\perp. \] But \(M \cap M^\perp = \{0\}\), so \(y_1 = y_2\) and \(z_1 = z_2\).

Conclusion: Every vector in a Hilbert space can be **uniquely decomposed** into a part in a closed subspace and a part in its orthogonal complement. The map \(x \mapsto y\) defines the **orthogonal projection** onto \(M\).

36: Bounded Linear Operators

Question: Let \(H\) be a Hilbert space. A linear operator \(T: H \to H\) is called bounded if there exists \(M \ge 0\) such that \[ \|Tx\| \le M \|x\| \quad \forall x \in H. \] Prove that every bounded linear operator is continuous. Also define the operator norm.

Solution / Proof

Definition: The operator norm of \(T\) is \[ \|T\| = \sup_{\|x\|=1} \|Tx\| = \sup_{x \neq 0} \frac{\|Tx\|}{\|x\|}. \]

Proof of continuity:

  1. Let \(T\) be bounded: \(\|Tx\| \le M \|x\|\) for all \(x \in H\).
  2. Take any sequence \(x_n \to x\) in \(H\). Then \[ \|T x_n - T x\| = \|T(x_n - x)\| \le M \|x_n - x\| \to 0. \]
  3. Hence \(T\) is continuous.

Conclusion: Boundedness and continuity are equivalent for linear operators on Hilbert spaces.

37: Adjoint Operators and Special Classes

Question: Let \(H\) be a Hilbert space and \(T: H \to H\) a bounded linear operator. Consider the following:

  1. Define the adjoint operator \(T^*\) and prove its existence and uniqueness.
  2. Define the following special classes of operators and give basic properties:
    • Self-adjoint operators
    • Unitary operators
    • Normal operators
    • Positive operators
Solution / Proof

Part 1: Adjoint Operators

Theorem: For every bounded linear operator \(T: H \to H\), there exists a unique bounded linear operator \(T^*: H \to H\) such that \[ \langle Tx, y \rangle = \langle x, T^* y \rangle \quad \forall x, y \in H. \]

Proof:

  1. Fix \(y \in H\) and define the linear functional \(f_y(x) := \langle Tx, y \rangle\). Then \(f_y \in H^*\) (continuous dual) since \(|f_y(x)| = |\langle Tx, y \rangle| \le \|Tx\| \|y\| \le \|T\| \|x\| \|y\|\).
  2. By the Riesz Representation Theorem, there exists a unique \(z \in H\) such that \[ f_y(x) = \langle x, z \rangle \quad \forall x \in H. \] Define \(T^* y := z\).
  3. Uniqueness follows from uniqueness in the Riesz representation. Linearity and boundedness of \(T^*\) also follow: \(\|T^*\| = \|T\|\).

Conclusion: Every bounded operator on a Hilbert space has a unique adjoint operator.

Part 2: Special Classes of Operators

  • Self-adjoint: \(T = T^*\). Properties: \(\langle Tx, x \rangle \in \mathbb{R}\), spectrum is real.
  • Unitary: \(T^* T = T T^* = I\). Properties: preserves norms, \(\|Tx\| = \|x\|\), and the spectrum lies on the unit circle.
  • Normal: \(T^* T = T T^*\). Properties: generalises self-adjoint/unitary; admits a spectral theorem.
  • Positive operators: \(T = T^*\) and \(\langle Tx, x \rangle \ge 0\) for all \(x \in H\). Properties: square roots exist, spectrum lies in \([0, \infty)\).

Remark: These classes are fundamental in spectral theory, quantum mechanics, and functional analysis.

38: Unitary Operators Preserve Inner Products and Norms

Question: Let \(U: H \to H\) be a unitary operator on a Hilbert space \(H\), i.e., \(U^* U = U U^* = I\). Prove that:

  1. \(\|Ux\| = \|x\|\) for all \(x \in H\) (norm-preserving).
  2. \(\langle Ux, Uy \rangle = \langle x, y \rangle\) for all \(x, y \in H\) (inner product-preserving).
Solution / Proof

Proposition:

If \(U\) is unitary (\(U^* U = I\)), then for all \(x, y \in H\):

  • \(\|Ux\| = \|x\|\)
  • \(\langle Ux, Uy \rangle = \langle x, y \rangle\)

Proof:

  1. Norm preservation: \[ \|Ux\|^2 = \langle Ux, Ux \rangle = \langle x, U^* U x \rangle = \langle x, x \rangle = \|x\|^2. \] Taking square roots gives \(\|Ux\| = \|x\|\).
  2. Inner product preservation: For any \(x, y \in H\): \[ \langle Ux, Uy \rangle = \langle x, U^* U y \rangle = \langle x, y \rangle. \]

Conclusion: Unitary operators preserve both norms and inner products, and thus preserve the Hilbert space structure. They are isometries of \(H\).

39: Isometries in Hilbert Spaces

Question: Define an isometry on a Hilbert space \(H\) and give an example. Explain why every unitary operator is an isometry.

Solution

Definition (Isometry):

A linear operator \(T: H \to H\) on a Hilbert space \(H\) is called an isometry if it preserves norms, i.e.,

\[ \|Tx\| = \|x\| \quad \forall x \in H. \]

Equivalently, \(T\) preserves the inner product: \(\langle Tx, Ty \rangle = \langle x, y \rangle\) for all \(x, y \in H\).

Example:

  • The identity operator \(I: H \to H\) is trivially an isometry.
  • Any unitary operator \(U\) is an isometry, since \(\|Ux\| = \|x\|\) and \(\langle Ux, Uy \rangle = \langle x, y \rangle\).

Remark: Every unitary operator is an isometry, but not every isometry is unitary (it may not be surjective).

Key Points: Spectral Theory for Bounded Operators

Let \(T: H \to H\) be a bounded linear operator on a Hilbert space \(H\).

Key Definitions:

  • Resolvent set: \(\rho(T) = \{\lambda \in \mathbb{C} : (T - \lambda I) \text{ is bijective and } (T - \lambda I)^{-1} \text{ is bounded} \}.\)
  • Spectrum: \(\sigma(T) = \mathbb{C} \setminus \rho(T) = \{\lambda \in \mathbb{C} : T - \lambda I \text{ is not invertible}\}.\)
  • Point spectrum: Eigenvalues of \(T\): \(\sigma_p(T) = \{\lambda \in \mathbb{C} : \exists x \neq 0, Tx = \lambda x \}\).
  • Continuous spectrum: \(\sigma_c(T) = \{\lambda \in \mathbb{C} : T - \lambda I \text{ injective, range dense, not surjective}\}.\)
  • Residual spectrum: \(\sigma_r(T) = \{\lambda \in \mathbb{C} : T - \lambda I \text{ injective, range not dense}\}.\)

Key Properties:

  • The spectrum \(\sigma(T)\) is a non-empty compact subset of \(\mathbb{C}\).
  • For any \(\lambda \notin \sigma(T)\), the resolvent operator \(R_\lambda(T) = (T - \lambda I)^{-1}\) exists and is bounded.
  • For a bounded operator, \(\sigma(T) \subset \{ \lambda \in \mathbb{C} : |\lambda| \le \|T\| \}\).
  • If \(T\) is compact, then every non-zero \(\lambda \in \sigma(T)\) is an eigenvalue with finite multiplicity, and \(0\) is the only possible accumulation point.
  • Spectrum of a self-adjoint operator is real; for unitary operators, it lies on the unit circle; for positive operators, in \([0, \infty)\).

Remark:

Spectral theory provides a decomposition of bounded operators and is fundamental for functional calculus, solving operator equations, and understanding operator behavior in Hilbert spaces. Compact operators behave like “infinite-dimensional matrices” with discrete spectra accumulating only at 0.

Key Points: Spectral Theorem for Bounded Self-Adjoint Operators

Let \(H\) be a Hilbert space and \(T: H \to H\) a bounded self-adjoint operator (\(T = T^*\)).

Key Idea:

Every bounded self-adjoint operator can be “diagonalised” in a Hilbert space sense using a projection-valued measure (PVM). This generalises the concept of eigenvectors and eigenvalues to infinite-dimensional spaces.

Definitions:

  • Projection-Valued Measure (PVM): A map \(E: \mathcal{B}(\mathbb{R}) \to \mathcal{B}(H)\) from the Borel sets of \(\mathbb{R}\) to bounded projections on \(H\) satisfying:
    1. \(E(\emptyset) = 0, E(\mathbb{R}) = I\)
    2. \(E(B_1 \cap B_2) = E(B_1) E(B_2)\)
    3. \(E(\bigcup_{n=1}^\infty B_n) = \sum_{n=1}^\infty E(B_n)\) for disjoint Borel sets (strong operator topology convergence)

Spectral Theorem (Bounded Self-Adjoint Version):

There exists a unique PVM \(E\) such that:

\[ T = \int_{\sigma(T)} \lambda \, dE(\lambda) \]

and for any bounded Borel function \(f: \sigma(T) \to \mathbb{C}\),

\[ f(T) = \int_{\sigma(T)} f(\lambda) \, dE(\lambda) \]

This is called the functional calculus for \(T\).

Properties:

  • \(\sigma(T) \subset \mathbb{R}\)
  • The PVM \(E(B)\) projects onto the subspace where \(T\) “takes values” in \(B \subset \mathbb{R}\)
  • If \(T\) has purely discrete spectrum, \(E\) reduces to projections onto eigenspaces
  • Allows defining functions of \(T\) rigorously

Intuition:

Think of \(T\) as an infinite-dimensional “diagonal matrix”: each \(\lambda \in \sigma(T)\) acts like an eigenvalue, and \(E(\{\lambda\})\) projects onto the corresponding subspace. The integral sums over all these “eigencomponents.”

Sketch of Proof

Step 1: Reduce to real C*-algebra:

The C*-algebra generated by \(T\) and \(I\) is commutative and self-adjoint.

Step 2: Apply Gelfand Representation:

For a commutative unital C*-algebra, there exists a *-isomorphism to continuous functions on a compact Hausdorff space. Here, the spectrum \(\sigma(T)\) plays that role.

Step 3: Construct the PVM:

Use the Riesz-Markov theorem to associate each Borel set \(B \subset \sigma(T)\) with a projection \(E(B)\) such that

\[ \langle x, E(B) x \rangle = \mu_x(B) \] for a measure \(\mu_x\) induced by the state \(x \mapsto \langle x, T x \rangle\).

Step 4: Express \(T\) as the integral:

Define the operator integral

\[ T = \int_{\sigma(T)} \lambda \, dE(\lambda), \] which recovers the action of \(T\) on every vector via \(\langle x, T x \rangle = \int \lambda \, d\mu_x(\lambda)\).

Remark: If \(T\) has discrete spectrum, the integral reduces to a sum over eigenvalues with orthogonal projections, connecting back to finite-dimensional diagonalisation.

1. Key Points: Functional Calculus for Bounded Self-Adjoint Operators

Theorem (Continuous Functional Calculus):

Let \(T: H \to H\) be a bounded self-adjoint operator on a Hilbert space \(H\) and let \(\sigma(T) \subset \mathbb{R}\) be its spectrum. For every continuous function \(f: \sigma(T) \to \mathbb{C}\), there exists a unique bounded operator \(f(T)\) such that:

  1. If \(f(\lambda) = \lambda\), then \(f(T) = T\).
  2. If \(f \equiv 1\), then \(f(T) = I\).
  3. The map \(f \mapsto f(T)\) is a *-homomorphism: \(\overline{f}(T) = f(T)^*\), \((fg)(T) = f(T) g(T)\).
  4. \(\|f(T)\| = \sup_{\lambda \in \sigma(T)} |f(\lambda)|\).

Example:

  • Take \(T\) self-adjoint and \(f(\lambda) = \lambda^2\). Then \(f(T) = T^2\).
  • For \(f(\lambda) = e^{i\lambda}\), \(f(T)\) defines a unitary operator \(e^{iT}\).

2. Compact Self-Adjoint Operators: Discrete Spectrum

Theorem: Let \(T: H \to H\) be a compact self-adjoint operator. Then:

  1. The spectrum \(\sigma(T)\) is at most countable.
  2. 0 is the only possible accumulation point of \(\sigma(T)\).
  3. All non-zero elements of \(\sigma(T)\) are eigenvalues of finite multiplicity.
  4. There exists an orthonormal basis of \(H\) consisting of eigenvectors of \(T\).

Proof:

  1. Step 1: Non-zero eigenvalues exist: If \(T \neq 0\), then by compactness, \(T\) maps the unit ball to a relatively compact set. Using the Riesz lemma, one shows the existence of \(\lambda \neq 0\) and \(x \neq 0\) such that \(Tx = \lambda x\).
  2. Step 2: Orthogonality of eigenvectors: If \(x\) and \(y\) are eigenvectors corresponding to distinct eigenvalues \(\lambda \neq \mu\), then \(\langle Tx, y \rangle = \lambda \langle x, y \rangle = \langle x, Ty \rangle = \mu \langle x, y \rangle\). Hence \(\langle x, y \rangle = 0\).
  3. Step 3: Discreteness of non-zero spectrum: Assume an infinite sequence of non-zero eigenvalues accumulating at a non-zero point. Compactness implies existence of a convergent subsequence in \(H\), leading to a contradiction. Hence, 0 is the only possible accumulation point.
  4. Step 4: Orthonormal basis of eigenvectors: Take the span of eigenvectors corresponding to non-zero eigenvalues and complete it with an orthonormal basis of \(\ker(T)\). This gives an orthonormal basis of \(H\) diagonalising \(T\).

Example:

  • Hilbert-Schmidt operators on \(L^2([0,1])\) are compact and self-adjoint. Their eigenfunctions form a countable orthonormal basis, e.g., Fourier sine/cosine series.

Remark: Compact self-adjoint operators behave like infinite-dimensional symmetric matrices: their “diagonalisation” generalises classical linear algebra, and functional calculus allows applying functions to these operators.

40: Operator Theory – Three Key Classes

Question: Let \(H\) be a Hilbert space (or a Banach space). Discuss the following classes of operators, including their definitions, properties, and examples.

  1. Bounded Linear Operators: Define a bounded linear operator \(T: H \to H\) and state its basic properties.
  2. Compact Operators: Define compact operators, describe their spectrum, and give an example.
  3. Fredholm Operators: Define a Fredholm operator and state the index formula. Give a basic example.
Solution / Notes

1. Bounded Linear Operators

Definition: A linear operator \(T: H \to H\) is bounded if there exists \(C \ge 0\) such that

\[ \|Tx\| \le C \|x\| \quad \forall x \in H. \]

Key Points:

  • \(\|T\| = \sup_{\|x\|=1} \|Tx\|\) defines the operator norm.
  • Boundedness \(\iff\) continuity.
  • Examples: identity \(I\), matrices in \(\mathbb{C}^n\), integral operators with bounded kernels.

2. Compact Operators

Definition: \(T: H \to H\) is compact if the image of the unit ball is relatively compact, i.e., every bounded sequence \((x_n)\) has a subsequence \((Tx_{n_k})\) converging in \(H\).

Key Properties:

  • Every compact operator is bounded.
  • Spectrum \(\sigma(T)\) is at most countable, with 0 as the only possible accumulation point.
  • Non-zero spectrum consists of eigenvalues with finite multiplicity.
  • Example: Hilbert-Schmidt operators, integral operators with square-integrable kernels.

3. Fredholm Operators

Definition: \(T: H \to H\) is a Fredholm operator if

  • \(\ker(T)\) is finite-dimensional
  • \(\operatorname{coker}(T) = H / \operatorname{ran}(T)\) is finite-dimensional
  • \(\operatorname{ran}(T)\) is closed

Index of a Fredholm operator:

\[ \mathrm{index}(T) = \dim \ker(T) - \dim \operatorname{coker}(T). \]

Example: \(T = I + K\), where \(K\) is compact, is Fredholm with index 0.

Key Points: Spectral Theory of Compact Operators

  • Definition: \(T: H \to H\) (Hilbert or Banach space) is compact if it maps bounded sets to relatively compact sets (i.e., every bounded sequence has a convergent subsequence).
  • Spectrum is discrete (except 0): - \(\sigma(T) \setminus \{0\}\) consists of eigenvalues with finite multiplicity. - 0 is the only possible accumulation point.
  • Eigenvectors form an orthonormal basis (Hilbert case): - There exists an orthonormal set of eigenvectors corresponding to non-zero eigenvalues. - Completing with a basis of \(\ker(T)\) gives a full orthonormal basis of \(H\).
  • Functional calculus: - Functions of compact self-adjoint operators can be expressed in terms of their eigenvalues and eigenvectors: \[ f(T) x = \sum_{n} f(\lambda_n) \langle x, e_n \rangle e_n \]
  • Adjoint: - If \(T\) is compact, \(T^*\) is also compact. - Non-zero spectra of \(T\) and \(T^*\) coincide.
  • Examples: - Hilbert-Schmidt operators (integral operators with square-integrable kernels) - Finite-rank operators - Compact perturbations of the identity: \(I + K\)
  • Importance: - Compact operators generalise finite-dimensional linear algebra to infinite dimensions. - Their spectral decomposition is fundamental in solving operator equations, PDEs, and functional analysis problems.

41: Riesz Projection

Question: Let \(T: X \to X\) be a bounded linear operator on a Banach space \(X\), and let \(\sigma(T)\) be its spectrum. Suppose \(\Gamma \subset \mathbb{C}\) is a closed contour in the resolvent set of \(T\) that encloses an isolated part of the spectrum \(\sigma_0 \subset \sigma(T)\). Define the Riesz projection corresponding to \(\sigma_0\) and state its key properties.

Solution / Notes

Definition (Riesz Projection):

The Riesz projection \(P\) corresponding to the isolated spectral subset \(\sigma_0\) is defined by:

\[ P = \frac{1}{2\pi i} \oint_{\Gamma} (zI - T)^{-1} \, dz \]

where \(\Gamma\) is a positively oriented contour enclosing \(\sigma_0\) and no other points of \(\sigma(T)\).

Key Properties:

  • Projection: \(P^2 = P\), i.e., \(P\) is idempotent.
  • Invariant subspace: \(\operatorname{ran}(P)\) and \(\ker(P)\) are invariant under \(T\).
  • Decomposition of \(T\): \[ T = T_0 \oplus T_1 \quad \text{on } \operatorname{ran}(P) \oplus \ker(P), \] where \(\sigma(T_0) = \sigma_0\) and \(\sigma(T_1) = \sigma(T) \setminus \sigma_0\).
  • Analyticity: \(P\) depends analytically on \(T\) in a neighborhood of \(T\) in operator norm.
  • Finite-dimensional case: If \(\sigma_0\) consists of eigenvalues of finite multiplicity, \(P\) projects onto the corresponding generalised eigenspace.

Intuition:

Think of the Riesz projection as an **infinite-dimensional analogue of projecting onto the eigenspace of selected eigenvalues**. The contour integral picks out the part of the operator associated only with \(\sigma_0\), separating it from the rest of the spectrum.

42: Riesz Decomposition Theorem

Question: Let \(T: X \to X\) be a bounded linear operator on a Banach space \(X\). Suppose \(\sigma_0 \subset \sigma(T)\) is an isolated part of the spectrum, and let \(P\) be the Riesz projection corresponding to \(\sigma_0\). State and prove the Riesz decomposition theorem for \(X\) with respect to \(\sigma_0\).

Solution / Notes / Proof Sketch

Theorem (Riesz Decomposition):

Let \(T: X \to X\) be bounded and \(\sigma_0 \subset \sigma(T)\) an isolated spectral set. Let \(P\) be the corresponding Riesz projection. Then:

  1. Direct sum decomposition: \[ X = \operatorname{ran}(P) \oplus \ker(P), \] where \(\operatorname{ran}(P)\) and \(\ker(P)\) are closed, \(T\)-invariant subspaces.
  2. Restricted spectra: \(\sigma(T|_{\operatorname{ran}(P)}) = \sigma_0\), \(\sigma(T|_{\ker(P)}) = \sigma(T) \setminus \sigma_0\).
  3. Operator decomposition: \[ T = T_0 \oplus T_1, \quad T_0 = T|_{\operatorname{ran}(P)}, \quad T_1 = T|_{\ker(P)}. \]

Proof Sketch:

  1. Step 1 – Define Riesz projection: \[ P = \frac{1}{2\pi i} \oint_{\Gamma} (zI - T)^{-1} dz, \] where \(\Gamma\) encloses \(\sigma_0\) but no other part of \(\sigma(T)\).
  2. Step 2 – Show \(P\) is a projection: By the properties of contour integrals and the resolvent, \(P^2 = P\) and \(PT = TP\). Hence \(\operatorname{ran}(P)\) and \(\ker(P)\) are \(T\)-invariant.
  3. Step 3 – Decompose \(X\): Every \(x \in X\) can be written uniquely as \[ x = Px + (I-P)x \in \operatorname{ran}(P) \oplus \ker(P). \]
  4. Step 4 – Restrict operator: On \(\operatorname{ran}(P)\), the resolvent formula shows that the spectrum is exactly \(\sigma_0\). On \(\ker(P)\), by construction, the spectrum is \(\sigma(T) \setminus \sigma_0\).

Remark / Intuition:

This theorem generalises **finite-dimensional spectral decomposition**: it “isolates” the part of the operator associated with a selected set of eigenvalues, even in infinite-dimensional Banach spaces. Riesz projections allow us to study these parts separately, which is especially useful in perturbation theory and spectral analysis.

Unbounded Operators on Hilert Space

Key Points

  • Domain is crucial: An unbounded operator \(T\) is a linear map \(T: D(T) \subset H \to H\), where \(D(T)\) is a dense subspace of \(H\). Unlike bounded operators, the domain is not necessarily all of \(H\).
  • Closed operators: \(T\) is closed if its graph \(\mathcal{G}(T) = \{(x, Tx) : x \in D(T)\}\) is closed in \(H \oplus H\).
  • Closable operators: \(T\) is closable if it has a closed extension \(\overline{T}\), i.e., the closure of the graph \(\mathcal{G}(T)\) is itself a graph of an operator.

43: Closed and Closable Operators

Question: Let \(T: D(T) \subset H \to H\) be a densely defined linear operator. Prove the following proposition:

  1. Proposition: \(T\) is closable if and only if for every sequence \((x_n) \subset D(T)\) with \(x_n \to 0\) and \(Tx_n \to y \in H\), we have \(y = 0\).
  2. Show that if \(T\) is closed, then it is automatically closable.
Solution / Proof Sketch

Proof:

  1. Step 1 – Necessity: Suppose \(T\) is closable with closure \(\overline{T}\). If \(x_n \to 0\) and \(Tx_n \to y\), then \((0, y) \in \mathcal{G}(\overline{T})\). Since \(\overline{T}\) is closed, its graph is closed, and the only element with 0 in the first component is 0 in the second. Hence \(y = 0\).
  2. Step 2 – Sufficiency: Suppose the condition holds. Let \(\mathcal{G}(\overline{T})\) be the closure of \(\mathcal{G}(T)\) in \(H \oplus H\). Define \(\overline{T}\) on the set of all \(x \in H\) such that \((x, y) \in \mathcal{G}(\overline{T})\). The uniqueness condition ensures that \(\overline{T}\) is well-defined and closed.
  3. Step 3 – Closed implies closable: If \(T\) is closed, then its graph is already closed, so trivially \(\overline{T} = T\). Hence every closed operator is closable.

Conclusion: This proposition gives a practical criterion to check whether a densely defined operator can be extended to a closed operator.

Key Points: Self-Adjoint Extensions

  • For symmetric operators \(T\) (\(\langle Tx, y \rangle = \langle x, Ty \rangle\) for all \(x, y \in D(T)\)), a key is whether \(T\) admits a self-adjoint extension.
  • Von Neumann’s theory: the existence of self-adjoint extensions is determined by the deficiency indices \((n_+, n_-) = (\dim \ker(T^* - iI), \dim \ker(T^* + iI))\).
  • If \(n_+ = n_-\), there exists a family of self-adjoint extensions of \(T\).
  • Examples: The momentum operator \(P = -i d/dx\) on \(L^2([0,1])\) with domain \(C_c^\infty(0,1)\) is symmetric but not self-adjoint; proper boundary conditions yield self-adjoint extensions.

Topological Vector Spaces (TVS)

Definition

A topological vector space is a vector space \(X\) over the field \(\mathbb{R}\) or \(\mathbb{C}\) equipped with a topology \(\tau\) such that:

  1. The vector addition \(+: X \times X \to X\), \((x,y) \mapsto x+y\), is continuous with respect to the product topology on \(X \times X\).
  2. The scalar multiplication \(\cdot: \mathbb{F} \times X \to X\), \((\alpha, x) \mapsto \alpha x\), is continuous with respect to the usual topology on \(\mathbb{F}\) and \(\tau\) on \(X\).

We denote the space as \((X, \tau)\), where \(X\) is the vector space and \(\tau\) is the topology.

Key Points / Remarks

  • No norm required: Unlike normed spaces, the topology \(\tau\) need not come from a norm. The only requirement is continuity of addition and scalar multiplication.
  • Examples:
    • Normed spaces (\(X, \|\cdot\|\)) with the metric topology from the norm.
    • The space \(C([0,1])\) with the topology of uniform convergence on compact sets (locally convex, but not necessarily normed in some versions).
    • Sequence spaces with the product topology, e.g., \(\mathbb{R}^\mathbb{N}\) with pointwise convergence.

    Definition: Seminorm

    Let \(X\) be a vector space over \(\mathbb{R}\) or \(\mathbb{C}\). A function \(p: X \to [0, \infty)\) is called a seminorm if it satisfies:

    1. Absolute homogeneity: \[ p(\alpha x) = |\alpha| \, p(x) \quad \forall \alpha \in \mathbb{F}, \, x \in X. \]
    2. Subadditivity (triangle inequality): \[ p(x + y) \le p(x) + p(y) \quad \forall x, y \in X. \]

    Remark: Unlike a norm, a seminorm may satisfy \(p(x) = 0\) for some non-zero \(x \in X\). If \(p(x) = 0 \implies x = 0\), then \(p\) is a norm.

    44: Locally Convex Topological Vector Spaces

    Question: Let \((X, \tau)\) be a topological vector space. Define what it means for the topology \(\tau\) to be locally convex and explain the characterisation in terms of seminorms.

    Solution / Answer

    Definition / characterisation:

    The topology \(\tau\) on \(X\) is locally convex if there exists a neighborhood base of \(0 \in X\) consisting of **convex sets**. Equivalently, \(\tau\) is locally convex if it can be generated by a family of seminorms \(\{p_i\}_{i \in I}\), meaning the open sets in \(X\) are unions of sets of the form:

    \[ U = \big\{ x \in X : p_{i_1}(x - x_0) < \epsilon_1, \dots, p_{i_n}(x - x_0) < \epsilon_n \big\} \]

    for some finite collection of seminorms \(p_{i_1}, \dots, p_{i_n}\), \(\epsilon_1, \dots, \epsilon_n > 0\), and \(x_0 \in X\). In other words, \(U \subset X\) is open if for every \(x \in U\), there exist finitely many seminorms \(p_{i_1}, \dots, p_{i_n}\) and \(\epsilon_1, \dots, \epsilon_n > 0\) such that:

    \[ \{y \in X : p_{i_k}(y - x) < \epsilon_k, \, k = 1, \dots, n \} \subset U. \]

    Remarks:

    • Every normed space is locally convex because the norm itself is a seminorm and generates convex balls.
    • Locally convex spaces are the natural setting for the **Hahn–Banach theorem** and for studying **dual spaces** in functional analysis.
    • Convexity of neighborhoods ensures good separation properties and allows the use of continuous linear functionals to study the topology.

    45: metrisability of a Topological Space

    Question: What does it mean for a topological space \((X, \tau)\) to be metrisable?

    Solution / Answer

    A topological space \((X, \tau)\) is said to be metrisable if there exists a metric \(d: X \times X \to [0, \infty)\) such that the topology induced by \(d\) coincides with \(\tau\). Equivalently:

    • Open sets in \((X, \tau)\) are exactly the unions of open balls \(B_d(x, r) = \{y \in X : d(x,y) < r\}\) for \(x \in X\) and \(r > 0\).
    • Every metrisable space is a Hausdorff space, and sequences behave nicely with respect to convergence (sequential convergence characterises the topology).

    Remark: In locally convex spaces, metrisability often requires the topology to be generated by a countable family of seminorms.

    46: Fréchet Spaces

    Definition

    A Fréchet space is a topological vector space \((X, \tau)\) over \(\mathbb{R}\) or \(\mathbb{C}\) satisfying:

    1. Locally convex: \(\tau\) has a neighborhood base of 0 consisting of convex sets (equivalently, the topology is generated by a family of seminorms \(\{p_n\}_{n \in \mathbb{N}}\)).
    2. metrisable: There exists a translation-invariant metric \(d\) generating \(\tau\).
    3. Complete: Every Cauchy sequence (with respect to the metric) converges in \(X\).

    Equivalently, a Fréchet space is a **complete, metrisable, locally convex topological vector space**.

    Key Properties

    • Can be described via a **countable family of seminorms** \(p_1, p_2, \dots\) such that a sequence \((x_n)\) converges if and only if \(p_k(x_n - x) \to 0\) for all \(k \in \mathbb{N}\).
    • All Banach spaces are Fréchet spaces (take the single norm as the seminorm family).
    • Spaces of smooth functions \(C^\infty([0,1])\) with seminorms \(p_n(f) = \sup_{x \in [0,1]} |f^{(n)}(x)|\) are Fréchet spaces but not Banach.
    • Fréchet spaces are **locally convex and complete**, making them suitable for functional analysis beyond Banach spaces.
    • metrisability implies sequential methods can be used to study convergence, compactness, etc.

    47: LF Spaces (Limit of Fréchet Spaces)

    Definition

    An LF space is a topological vector space \(X\) that is the **inductive limit of a sequence of Fréchet spaces**:

    \[ X = \bigcup_{n=1}^\infty X_n, \quad X_1 \subset X_2 \subset X_3 \subset \cdots \] where each \(X_n\) is a Fréchet space and the inclusion maps \(X_n \hookrightarrow X_{n+1}\) are continuous. The topology on \(X\) is the **finest locally convex topology** such that each inclusion \(X_n \to X\) is continuous.

    Key Points / Properties

    • LF spaces are **locally convex** but generally **not metrisable** (unless the sequence stabilises at some \(X_n\)).
    • They are **complete** if each \(X_n\) is complete (i.e., Fréchet).
    • They allow the study of spaces that are “too big” to be Fréchet, e.g., spaces of **test functions**:
      • \(\mathcal{D}(\Omega) = C_c^\infty(\Omega)\), the space of smooth functions with compact support on an open set \(\Omega \subset \mathbb{R}^n\), is an LF space.
      • Used in **distribution theory**, because distributions are continuous linear functionals on LF spaces.
    • Sequential convergence is **more subtle**, often requiring the convergence to occur in some stage \(X_n\).

    48: Distribution Spaces

    Definition: Space of Distributions

    Let \(\Omega \subset \mathbb{R}^n\) be open. The space of test functions with compact support is denoted by:

    \[ \mathcal{D}(\Omega) := C_c^\infty(\Omega) \]

    equipped with its standard LF topology (inductive limit of Fréchet spaces). Then the space of distributions on \(\Omega\) is defined as the topological dual:

    \[ \mathcal{D}'(\Omega) := \{ T: \mathcal{D}(\Omega) \to \mathbb{C} \mid T \text{ is linear and continuous} \}. \]

    Key Properties

    • Locally convex: \(\mathcal{D}'(\Omega)\) is a locally convex space as the dual of a locally convex space.
    • Complete: \(\mathcal{D}'(\Omega)\) is complete in the weak-* topology (Banach–Alaoglu style argument for LF spaces).
    • Not normable: There is no single norm generating the topology; convergence is defined via seminorms or bounded sets in \(\mathcal{D}(\Omega)\).
    • Dual topologies: There are two important notions of convergence for distributions:
      • Weak-* topology: \(T_\alpha \to T\) if \(\langle T_\alpha, \phi \rangle \to \langle T, \phi \rangle\) for all \(\phi \in \mathcal{D}(\Omega)\). This is pointwise convergence on test functions.
      • Strong dual topology: Uniform convergence on bounded subsets of \(\mathcal{D}(\Omega)\). Provides finer control, useful for continuity of operations like differentiation and convolution.
    • LF nature of \(\mathcal{D}(\Omega)\) ensures distributions can act on “large” spaces, but convergence in \(\mathcal{D}'(\Omega)\) can be subtle depending on the topology considered.

    Remarks

    • \(\mathcal{D}'(\Omega)\) contains all locally integrable functions as regular distributions, i.e., \(f \mapsto T_f\) with \(T_f(\phi) = \int_\Omega f(x) \phi(x) dx\).
    • The weak-* topology is useful for defining sequential convergence and continuity of linear functionals.
    • The strong dual topology is necessary to guarantee continuity of operations that involve families of test functions simultaneously.
    • Distribution spaces are a prototypical example of LF duals: **non-normable, complete, locally convex, and metrisable on bounded sets**.

    48: Duality in Locally Convex Spaces

    Question: Let \(X\) be a locally convex space (LCS) over \(\mathbb{R}\) or \(\mathbb{C}\). Consider its continuous dual \(X'\) (the space of all continuous linear functionals on \(X\)).

    1. Define the weak-* topology
    2. Define the strong dual topology on \(X'\) and explain how convergence differs from the weak-* topology.
    3. Explain why the strong dual is generally finer than the weak-* topology and give an example where the two topologies differ.
    Solution / Answer

    1. Weak-* Topology on \(X'\)

    The weak-* topology on \(X'\) is the coarsest topology making all evaluation maps continuous:

    \[ \phi \mapsto \phi(x) \in \mathbb{F}, \quad \forall x \in X. \]

    Convergence in the weak-* topology: a net \((\phi_\alpha) \subset X'\) converges to \(\phi \in X'\) if and only if

    \[ \phi_\alpha(x) \to \phi(x) \quad \forall x \in X. \]

    Intuition: pointwise convergence on \(X\).

    2. Strong Dual Topology on \(X'\)

    The strong dual topology on \(X'\) is generated by the family of seminorms:

    \[ p_B(\phi) := \sup_{x \in B} |\phi(x)|, \quad \text{for all bounded sets } B \subset X. \]

    Convergence in the strong dual: a net \((\phi_\alpha)\) converges to \(\phi\) if and only if it converges uniformly on all bounded subsets of \(X\). This is stronger than pointwise convergence.

    3. Comparison and Example

    • The strong dual topology is **finer** than the weak-* topology because uniform convergence on bounded sets implies pointwise convergence on individual elements.
    • Example: Let \(X = \mathcal{D}(\Omega)\). Then \(\mathcal{D}'(\Omega)\) with the weak-* topology is pointwise convergence on test functions, whereas the strong dual requires uniform convergence on bounded subsets. These topologies are not the same, but both make \(\mathcal{D}'(\Omega)\) locally convex and complete.
    • Key consequence: operations like differentiation or convolution are continuous in the strong dual topology but may fail to be continuous in the weak-* topology.

    Summary: Weak-* = pointwise convergence, Strong = uniform on bounded sets. Strong dual is generally finer, capturing more “continuity control” over the LCS.

    49: Barrelled and Bornological Spaces

    Question: Let \(X\) be a locally convex space (LCS). Define and explain the following classes of spaces:

    1. Barrelled space: Define what it means for a locally convex space to be barrelled and explain the significance.
    2. Bornological space: Define what it means for a locally convex space to be bornological and explain the significance.
    Solution / Answer

    1. Barrelled Spaces

    A locally convex space \(X\) is called barrelled if every barrel in \(X\) is a neighborhood of 0. A barrel is a subset of \(X\) that is:

    • Closed
    • Convex
    • Balanced (absorbs scalar multiples with |α| ≤ 1)
    • Absorbing (for every x in X, there exists t > 0 such that x ∈ tB)

    Significance: In barrelled spaces, every linear map from \(X\) to a Banach space that is **pointwise bounded** is automatically **continuous** (Banach–Steinhaus theorem / Uniform Boundedness Principle holds).

    2. Bornological Spaces

    A locally convex space \(X\) is called bornological if every bounded linear map from \(X\) into any locally convex space is automatically **continuous**. Equivalently:

    • Every bounded set in \(X\) is contained in the absolutely convex hull of some neighborhood of 0.
    • Bornological spaces allow **boundedness to control continuity**, which is useful for unbounded operators and inductive limit spaces.

    Examples / Remarks:

    • Every **Banach space** is barrelled and bornological.
    • LF spaces, like \(\mathcal{D}(\Omega)\), are bornological but not necessarily barrelled.
    • Barrelled vs bornological: barrelled controls continuity via **barrels (large sets)**, bornological controls continuity via **bounded sets**.

    50: σ-Algebras

    Question: Let \(X\) be a set. Define what a σ-algebra on \(X\) is, and state its key properties.

    Solution / Answer

    A σ-algebra \(\mathcal{F}\) on a set \(X\) is a collection of subsets of \(X\) satisfying the following properties:

    1. Non-empty: \(X \in \mathcal{F}\).
    2. Closed under complements: If \(A \in \mathcal{F}\), then \(X \setminus A \in \mathcal{F}\).
    3. Closed under countable unions: If \(A_1, A_2, \dots \in \mathcal{F}\), then \(\bigcup_{n=1}^{\infty} A_n \in \mathcal{F}\).

    From these axioms, it follows that σ-algebras are also closed under:

    • Countable intersections: \(\bigcap_{n=1}^{\infty} A_n \in \mathcal{F}\)
    • Set differences: \(A \setminus B \in \mathcal{F}\) if \(A, B \in \mathcal{F}\)

    These properties make σ-algebras suitable as domains for defining measures.

    51: Measure and Its Properties

    Question: Let \((X, \mathcal{F})\) be a measurable space. Define what a measure is on \((X, \mathcal{F})\) and state its main properties.

    Solution / Answer

    A measure \(\mu\) on a measurable space \((X, \mathcal{F})\) is a function

    \[ \mu: \mathcal{F} \to [0, \infty] \]

    that satisfies the following properties:

    1. Non-negativity: For all \(A \in \mathcal{F}\), \(\mu(A) \ge 0\).
    2. Null empty set: \(\mu(\emptyset) = 0\).
    3. Countable additivity (σ-additivity): For any countable collection of pairwise disjoint sets \(\{A_n\} \subset \mathcal{F}\),
    \[ \mu\Big(\bigcup_{n=1}^{\infty} A_n \Big) = \sum_{n=1}^{\infty} \mu(A_n) \]

    From these axioms, it follows automatically that measures are:

    • Monotone: If \(A \subset B\), then \(\mu(A) \le \mu(B)\).
    • Countably subadditive: For any countable collection \(\{A_n\} \subset \mathcal{F}\),
    \[ \mu\Big(\bigcup_{n=1}^{\infty} A_n \Big) \le \sum_{n=1}^{\infty} \mu(A_n) \]

    These properties form the foundation for integration and probability theory.

    52: Lebesgue Integral – Definition and Steps

    Question: Let \((X, \mathcal{F}, \mu)\) be a measure space and \(f: X \to \mathbb{R}\) a measurable function. Describe the steps to define the Lebesgue integral of \(f\), including how to handle positive and negative parts.

    Solution / Answer

    Steps to Define the Lebesgue Integral

    1. Step 1: Non-negative simple functions: Start with a measurable function \(\phi: X \to [0, \infty)\) that takes only finitely many values (simple function). Define the integral:
      2. \[ \int_X \phi \, d\mu := \sum_{i=1}^{n} a_i \, \mu(A_i) \] where \(\phi = \sum_{i=1}^{n} a_i \chi_{A_i}\) and \(A_i\) are measurable, pairwise disjoint sets.
    2. Step 2: Extend to non-negative measurable functions: For \(f \ge 0\) measurable, define
      3. \[ \int_X f \, d\mu := \sup \left\{ \int_X \phi \, d\mu : 0 \le \phi \le f, \, \phi \text{ simple} \right\}. \]
    3. Step 3: Split general measurable functions: For a real-valued \(f\), write
      4. \[ f = f^+ - f^- \quad \text{where} \quad f^+ = \max(f,0), \quad f^- = \max(-f,0). \] Define \[ \int_X f \, d\mu := \int_X f^+ \, d\mu - \int_X f^- \, d\mu, \] provided at least one of \(\int_X f^+ d\mu\) or \(\int_X f^- d\mu\) is finite.
    4. Step 4: Absolute integrability: \(f\) is called integrable if \(\int_X |f| \, d\mu < \infty\). Then the integral is well-defined and finite.

    Summary: Lebesgue integration starts from simple non-negative functions, extends to all non-negative measurable functions by supremum, then splits a general function into positive and negative parts.

    53: Fundamental Theorems of Lebesgue Integration

    Question: Let \((X, \mathcal{F}, \mu)\) be a measure space. Prove the following key results of Lebesgue integration:

    1. Monotone Convergence Theorem (MCT)
    2. Fatou’s Lemma
    3. Dominated Convergence Theorem (DCT)
    Solution / Proof

    1. Monotone Convergence Theorem (MCT)

    Theorem: Let \((f_n)\) be a sequence of non-negative measurable functions on \(X\) such that \(f_n \uparrow f\) pointwise (i.e., \(f_1 \le f_2 \le \dots\) and \(f_n \to f\)). Then

    \[ \lim_{n \to \infty} \int_X f_n \, d\mu = \int_X f \, d\mu. \]

    Proof:

    1. Since \(f_n \le f\), the sequence \(\int f_n d\mu\) is **monotone increasing** and bounded above by \(\int f d\mu\) (possibly \(+\infty\)).
    2. By the definition of the Lebesgue integral for non-negative functions,
      3. \[ \int_X f \, d\mu = \sup \left\{ \int_X \phi \, d\mu : 0 \le \phi \le f, \phi \text{ simple} \right\}. \]
    3. Given any simple function \(\phi \le f\), there exists \(N\) such that \(\phi \le f_n\) for all \(n \ge N\). Hence, \(\int \phi \le \int f_n\).
    4. Taking supremum over all \(\phi\), we get \(\lim_{n\to\infty} \int f_n = \int f\).
    ---

    2. Fatou’s Lemma

    Theorem: Let \((f_n)\) be a sequence of non-negative measurable functions. Then

    \[ \int_X \liminf_{n \to \infty} f_n \, d\mu \le \liminf_{n \to \infty} \int_X f_n \, d\mu. \]

    Proof:

    1. Define \(g_k := \inf_{n \ge k} f_n \). Then \(g_k \uparrow \liminf_{n \to \infty} f_n\).
    2. Apply MCT to \(g_k\):
      3. \[ \int_X \liminf_{n\to\infty} f_n \, d\mu = \lim_{k \to \infty} \int_X g_k \, d\mu. \]
    3. For each \(k\), \(g_k \le f_n\) for all \(n \ge k\), hence \(\int g_k \le \inf_{n \ge k} \int f_n\).
    4. Take \(k \to \infty\), then \(\lim_{k \to \infty} \int g_k \le \liminf_{n \to \infty} \int f_n\), proving Fatou's lemma.
    ---

    3. Dominated Convergence Theorem (DCT)

    Theorem: Let \((f_n)\) be a sequence of measurable functions such that \(f_n \to f\) pointwise and there exists an **integrable function** \(g \ge 0\) with \(|f_n| \le g\) for all \(n\). Then \(f\) is integrable and

    \[ \lim_{n \to \infty} \int_X f_n \, d\mu = \int_X f \, d\mu. \]

    Proof:

    1. Define \(h_n := g - f_n\) and \(k_n := g + f_n\), which are non-negative and satisfy \(h_n \to g - f\), \(k_n \to g + f\).
    2. Apply Fatou's lemma to \(h_n\) and \(k_n\):
      3. \[ \int_X (g - f) \, d\mu \le \liminf \int_X (g - f_n) \, d\mu, \quad \int_X (g + f) \, d\mu \le \liminf \int_X (g + f_n) \, d\mu. \]
    3. Rewriting gives:
      4. \[ \limsup \int_X f_n \le \int_X f \le \liminf \int_X f_n. \]
    4. Hence \(\lim \int f_n = \int f\), proving DCT.

    Conclusion: These three results provide the foundation for **passing limits under the integral** and handling sequences of measurable functions.

    Lp Spaces

    Key Points:

    • Let \((X, \mathcal{F}, \mu)\) be a measure space and \(1 \le p < \infty\). Then
      • \[ L^p(X, \mu) := \left\{ f : X \to \mathbb{R} \text{ (or } \mathbb{C}) \text{ measurable } : \int_X |f|^p \, d\mu < \infty \right\}. \]
    • Norm on \(L^p\):
      • \[ \|f\|_p := \left( \int_X |f|^p \, d\mu \right)^{1/p}. \]
    • For \(p = \infty\), define the essentially bounded functions:
      • \[ L^\infty(X, \mu) := \{ f : X \to \mathbb{R} \text{ measurable } : \text{ess sup}_{x \in X} |f(x)| < \infty \}, \] with norm \(\|f\|_\infty := \text{ess sup}_{x \in X} |f(x)|.\)
    • Important properties:
      • \(L^p\) is a **Banach space** under \(\|\cdot\|_p\).
      • For \(1 < p < \infty\), the dual space \((L^p)^* \cong L^q\) where \(1/p + 1/q = 1\).
      • \(L^2\) is a **Hilbert space** with inner product \(\langle f, g \rangle = \int_X f \overline{g} \, d\mu\).

    53: Key Properties of Lp Spaces: Hölder and Minkowski Inequalities

    Question: Prove Hölder’s and Minkowski’s inequalities for \(L^p(X, \mu)\), and show that the Lp norm satisfies the triangle inequality.

    Solution / Proof

    1. Hölder’s Inequality

    Theorem: Let \(1 < p, q < \infty\) with \(1/p + 1/q = 1\), and \(f \in L^p, g \in L^q\). Then

    \[ \int_X |f g| \, d\mu \le \|f\|_p \, \|g\|_q. \]

    Proof:

    1. Use Young’s inequality: for \(a, b \ge 0\), \(ab \le \frac{a^p}{p} + \frac{b^q}{q}\).
    2. Set \(a = |f(x)| / \|f\|_p\), \(b = |g(x)| / \|g\|_q\). Then
      3. \[ |f(x) g(x)| \le \frac{|f(x)|^p}{p \|f\|_p^p} + \frac{|g(x)|^q}{q \|g\|_q^q}. \]
    3. Integrate both sides over \(X\):
      4. \[ \int_X \frac{|f g|}{\|f\|_p \|g\|_q} \, d\mu \le \frac{1}{p} + \frac{1}{q} = 1 \quad \implies \quad \int_X |fg| \, d\mu \le \|f\|_p \|g\|_q. \]
    ---

    2. Minkowski’s Inequality (Triangle Inequality)

    Theorem: For \(f, g \in L^p(X, \mu)\), \(1 \le p < \infty\),

    \[ \|f + g\|_p \le \|f\|_p + \|g\|_p. \]

    Proof:

    1. For \(p = 1\), obvious from linearity of integral.
    2. For \(1 < p < \infty\), write
      3. \[ \|f + g\|_p^p = \int_X |f + g|^p \, d\mu. \]
    3. Use the convexity inequality: \(|f + g|^p \le |f| |f + g|^{p-1} + |g| |f + g|^{p-1}\).
    4. Apply Hölder’s inequality to each term with exponents \(p\) and \(q\):
      5. \[ \int_X |f| |f + g|^{p-1} \le \|f\|_p \, \||f + g|^{p-1}\|_q = \|f\|_p \, \|f + g\|_p^{p-1}, \] and similarly for \(g\).
    5. Combine terms: \(\|f + g\|_p^p \le (\|f\|_p + \|g\|_p) \|f + g\|_p^{p-1} \implies \|f + g\|_p \le \|f\|_p + \|g\|_p\).
    ---

    3. Lp Norm Properties

    • Non-negativity: \(\|f\|_p \ge 0\) and \(\|f\|_p = 0 \iff f = 0 \ \mu\text{-a.e.}\)
    • Homogeneity: \(\|\alpha f\|_p = |\alpha| \, \|f\|_p\)
    • Triangle inequality: \(\|f + g\|_p \le \|f\|_p + \|g\|_p\) (Minkowski)

    Therefore, \((L^p(X, \mu), \|\cdot\|_p)\) is a **normed space**.

    Duality of Lp Spaces

    Question: Let \((X, \mathcal{F}, \mu)\) be a measure space, and let \(1 < p < \infty\) with conjugate exponent \(q\) defined by \(1/p + 1/q = 1\). Describe the dual space of \(L^p(X, \mu)\) and prove it.

    Solution / Proof

    Proposition (Dual of Lp):

    For \(1 < p < \infty\), the dual space \((L^p(X, \mu))^*\) is isometrically isomorphic to \(L^q(X, \mu)\). Explicitly, for each \(g \in L^q\), the linear functional

    \[ \Lambda_g(f) := \int_X f(x) g(x) \, d\mu \] is continuous on \(L^p\) with operator norm \(\|\Lambda_g\| = \|g\|_q\), and every continuous linear functional on \(L^p\) arises this way. ---

    Proof:

    1. Step 1: Linearity and boundedness: For \(f \in L^p\), define \(\Lambda_g(f) = \int f g \, d\mu\). By Hölder’s inequality:
      2. \[ |\Lambda_g(f)| = \left|\int_X f g \, d\mu\right| \le \|f\|_p \, \|g\|_q, \] so \(\Lambda_g\) is continuous with \(\|\Lambda_g\| \le \|g\|_q\).
    2. Step 2: Norm equality: Take \(f = \frac{|g|^{q-1} \operatorname{sgn}(g)}{\|g\|_q^{q/p}} \in L^p\) (normalised), then \[ |\Lambda_g(f)| = \|g\|_q. \] Thus \(\|\Lambda_g\| = \|g\|_q\).
    3. Step 3: Surjectivity: Let \(\Lambda \in (L^p)^*\). Using the **Radon–Nikodym theorem**, there exists \(g \in L^q\) such that \[ \Lambda(f) = \int_X f g \, d\mu \quad \forall f \in L^p. \]
    4. Hence every continuous linear functional on \(L^p\) is represented by some \(g \in L^q\), completing the duality.

    Conclusion: \((L^p(X, \mu))^* \cong L^q(X, \mu)\) isometrically, which is fundamental in functional analysis for understanding bounded linear functionals, reflexivity of \(L^p\) spaces, and weak convergence.

    54: Radon–Nikodym Theorem

    Question: Let \(\nu\) and \(\mu\) be σ-finite measures on \((X, \mathcal{F})\) with \(\nu \ll \mu\) (ν absolutely continuous w.r.t μ). Prove that there exists a measurable function \(f: X \to [0, \infty)\) such that

    \[ \nu(E) = \int_E f \, d\mu \quad \forall E \in \mathcal{F}. \] ---
    Solution / Proof Sketch

    Theorem (Radon–Nikodym):

    If \(\nu \ll \mu\) are σ-finite measures, there exists a unique \(f \in L^1(\mu)\) such that

    \[ \nu(E) = \int_E f \, d\mu \quad \forall E \in \mathcal{F}. \] The function \(f\) is called the Radon–Nikodym derivative of ν with respect to μ, denoted \(f = \frac{d\nu}{d\mu}\).

    ---

    Proof Sketch:

    1. Step 1: σ-finite reduction: Write \(X = \bigcup_{n=1}^\infty X_n\) with \(\mu(X_n), \nu(X_n) < \infty\). It suffices to construct \(f\) on each \(X_n\) and then glue them together.
    2. Step 2: Define candidate function: Consider the set of all non-negative measurable functions \(\phi\) with \(\int_E \phi \, d\mu \le \nu(E)\) for all measurable \(E\). Let \[ f := \sup \{ \phi : \phi \le \nu \text{ in this sense} \}. \]
    3. Step 3: Use monotone convergence: Construct an increasing sequence \(\phi_n \uparrow f\). By Monotone Convergence Theorem, \[ \int_E f \, d\mu = \lim_{n \to \infty} \int_E \phi_n \, d\mu \le \nu(E). \]
    4. Step 4: Maximality argument: If there existed a set \(A\) with \(\int_A f \, d\mu < \nu(A)\), one could increase \(f\) on \(A\) to contradict the definition of \(f\) as a supremum. Hence equality holds: \[ \int_E f \, d\mu = \nu(E) \quad \forall E. \]
    5. Step 5: Uniqueness: If \(g\) also satisfies \(\int_E g \, d\mu = \nu(E)\), then \(\int_E (f-g) \, d\mu = 0\) for all \(E\), implying \(f = g\) μ-a.e.

    Conclusion: The Radon–Nikodym derivative \(f = \frac{d\nu}{d\mu}\) exists, is unique μ-a.e., and allows expressing any absolutely continuous measure w.r.t μ as an integral against μ.

    55: Weak Convergence in Lp Spaces

    Question: Define weak convergence in \(L^p(X, \mu)\) for \(1 < p < \infty\), state its key properties, and give an illustrative example.

    Solution / Notes

    Definition:

    A sequence \((f_n) \subset L^p(X, \mu)\) is said to converge weakly to \(f \in L^p(X, \mu)\), denoted \(f_n \rightharpoonup f\), if

    \[ \int_X f_n(x) g(x) \, d\mu \longrightarrow \int_X f(x) g(x) \, d\mu \quad \forall g \in L^q(X, \mu), \] where \(q\) is the conjugate exponent of \(p\) (\(1/p + 1/q = 1\)). ---

    Key Properties:

    • Weak convergence implies boundedness in norm: \(\sup_n \|f_n\|_p < \infty\).
    • Weak limits are unique.
    • Norm is lower semicontinuous under weak convergence: \(\|f\|_p \le \liminf_{n \to \infty} \|f_n\|_p\).
    • Every bounded sequence in \(L^p\), \(1 < p < \infty\), has a weakly convergent subsequence (Banach–Alaoglu theorem + reflexivity).
    ---

    Example:

    Let \(X = [0,1]\), \(f_n(x) = \sin(2 \pi n x)\) in \(L^2([0,1])\). Then \(\|f_n\|_2 = 1\), but for any \(g \in L^2([0,1])\),

    \[ \int_0^1 f_n(x) g(x) \, dx \to 0 \quad \text{as } n \to \infty. \]

    Hence \(f_n \rightharpoonup 0\) weakly, but \(\|f_n - 0\|_2 = 1 \not\to 0\). Intuition: weak convergence tests only against linear functionals (duals), not full norms.

    56: Advanced Spectral Theory: Unbounded Operators

    / Key Points: Let \(T\) be a (possibly unbounded) self-adjoint operator on a Hilbert space \(H\). Introduce the spectral measure and its properties, and describe the spectral theorem for unbounded operators.

    Solution / Notes

    1. Spectral Measure for Self-Adjoint Operators

    Let \(B(\mathbb{R})\) denote the Borel σ-algebra of \(\mathbb{R}\). A spectral measure associated with a self-adjoint operator \(T\) is a map

    \[ E : B(\mathbb{R}) \to \mathcal{B}(H) \quad \text{(bounded operators on } H) \] satisfying:
    1. Projection-valued: \(E(B)\) is an orthogonal projection for each Borel set \(B \subset \mathbb{R}\).
    2. Countable additivity: for disjoint \(B_1, B_2, \dots \in B(\mathbb{R})\), \(\sum_n E(B_n) = E(\bigcup_n B_n)\) (strong operator topology).
    3. Normalisation: \(E(\mathbb{R}) = I\), the identity operator.
    4. Compatibility with \(T\): for \(x, y \in D(T)\), \(\langle x, T y \rangle = \int_\mathbb{R} \lambda \, d \langle x, E(\lambda) y \rangle\).
    ---

    2. Spectral Theorem (Unbounded Case)

    Let \(T\) be self-adjoint on \(H\). Then there exists a unique spectral measure \(E\) such that

    \[ T = \int_\mathbb{R} \lambda \, dE(\lambda), \] in the sense that for all \(x \in D(T)\), the integral \(\int_\mathbb{R} \lambda^2 \, d\|E(\lambda)x\|^2 < \infty\) and \[ \langle x, T x \rangle = \int_\mathbb{R} \lambda \, d\|E(\lambda)x\|^2. \]

    This generalises the diagonalisation of bounded self-adjoint operators to unbounded operators.

    ---

    3. Properties of the Spectral Measure \(E\)

    • Support of \(E\) equals the spectrum of \(T\): \(\sigma(T) = \{\lambda \in \mathbb{R} : E(U) \neq 0 \ \forall U \ni \lambda \}\).
    • Functional calculus: For any Borel function \(f : \mathbb{R} \to \mathbb{C}\), \[ f(T) := \int_\mathbb{R} f(\lambda) \, dE(\lambda) \] defines a (possibly unbounded) operator on \(H\).
    • Strongly commuting projections: \(E(B_1) E(B_2) = E(B_1 \cap B_2)\).
    ---

    4. Intuition:

    Think of \(E(B)\) as “projecting onto the subspace of vectors with spectral values in \(B\).” Unbounded operators (like momentum or position in quantum mechanics) are not bounded, but the spectral theorem allows a **rigorous decomposition** into “generalised eigenvectors” via the spectral measure.

    57: Operator-Valued Riemann Sums and Spectral Integrals

    Question: Let \(T\) be a self-adjoint operator on a Hilbert space \(H\) with spectral measure \(E\). Define the operator-valued Riemann sums for approximating the spectral integral

    \[ T = \int_\mathbb{R} \lambda \, dE(\lambda). \]
    Solution / Definition

    1. Partition of the spectrum:

    Let \(\sigma(T) \subset \mathbb{R}\) be the spectrum of \(T\). Choose a partition

    \[ -\infty < \lambda_0 < \lambda_1 < \dots < \lambda_n < \infty \] of the real line (or a bounded interval containing the spectrum). For each subinterval \([\lambda_{k-1}, \lambda_k)\), pick a sample point \(\xi_k \in [\lambda_{k-1}, \lambda_k)\).

    ---

    2. Operator-valued Riemann sum:

    The Riemann sum approximating the spectral integral is

    \[ S_n := \sum_{k=1}^n \xi_k \, E([\lambda_{k-1}, \lambda_k)). \]

    Here each term \(\xi_k E([\lambda_{k-1}, \lambda_k))\) is a bounded operator on \(H\), and the sum \(S_n\) is also an operator on \(H\).

    ---

    3. Limit to spectral integral:

    The spectral integral is defined as the limit (in the strong operator topology or in norm if bounded):

    \[ \int_\mathbb{R} \lambda \, dE(\lambda) := \lim_{\| \Delta \| \to 0} \sum_{k=1}^n \xi_k E([\lambda_{k-1}, \lambda_k)), \] where \(\| \Delta \| = \max_k (\lambda_k - \lambda_{k-1})\) is the mesh of the partition.

    This is analogous to the classical Riemann integral, but each “area element” is replaced by a **projection operator multiplied by the sample point**.

    ---

    4. Intuition:

    • Think of \(E([\lambda_{k-1}, \lambda_k))\) as projecting onto the subspace of \(H\) corresponding to spectral values in that interval.
    • The Riemann sum approximates \(T\) by summing the contributions from each spectral slice.
    • In the limit of finer partitions, the sum converges to the full operator \(T\) in the strong operator topology.

    Self-Adjoint Operators and Spectral Measures

    Question: Let \(A\) be a self-adjoint operator on a Hilbert space \(H\). State and explain the existence and uniqueness of its spectral measure \(E_A\).

    Solution / Notes

    Theorem (Spectral Theorem for Self-Adjoint Operators)

    Let \(A\) be a (possibly unbounded) self-adjoint operator on a Hilbert space \(H\). Then there exists a unique **projection-valued measure**

    \[ E_A : B(\mathbb{R}) \to \mathcal{B}(H) \] such that:
    1. For each Borel set \(B \subset \mathbb{R}\), \(E_A(B)\) is an orthogonal projection on \(H\).
    2. Countable additivity: If \(B_1, B_2, \dots\) are disjoint Borel sets, then \(\sum_{n} E_A(B_n) = E_A(\bigcup_n B_n)\) in the strong operator topology.
    3. Normalisation: \(E_A(\mathbb{R}) = I_H\).
    4. Representation of \(A\) as a spectral integral: \[ A = \int_{\mathbb{R}} \lambda \, dE_A(\lambda), \] meaning that for all \(x \in D(A)\), \(\langle x, A x \rangle = \int_{\mathbb{R}} \lambda \, d\langle x, E_A(\lambda) x \rangle\).
    5. Uniqueness: No other projection-valued measure satisfies all the above properties for \(A\).
    ---

    Intuition:

    • \(E_A(B)\) “projects” vectors onto the subspace of \(H\) corresponding to spectral values of \(A\) lying in \(B\).
    • Even for unbounded operators (like momentum in quantum mechanics), this gives a rigorous decomposition “like diagonalisation”.
    • The spectral measure allows defining \(f(A)\) for Borel functions \(f\) via the **functional calculus**: \[ f(A) := \int_{\mathbb{R}} f(\lambda) \, dE_A(\lambda). \]

    58: Functional Calculus for Self-Adjoint Operators

    Question: Let \(A\) be a self-adjoint operator on a Hilbert space \(H\) with spectral measure \(E_A\). Define \(f(A)\) for a Borel measurable function \(f : \mathbb{R} \to \mathbb{C}\) and explain the main properties.

    Solution / Notes

    1. Definition (Borel Functional Calculus)

    Let \(f : \mathbb{R} \to \mathbb{C}\) be a **Borel measurable function**. Then the operator \(f(A)\) is defined by the spectral integral:

    \[ f(A) := \int_{\mathbb{R}} f(\lambda) \, dE_A(\lambda). \]

    This means that for each \(x \in H\),

    \[ \langle x, f(A) x \rangle = \int_{\mathbb{R}} f(\lambda) \, d\langle x, E_A(\lambda) x \rangle. \]

    If \(f\) is bounded, \(f(A)\) is a bounded operator. If \(f\) is unbounded, \(f(A)\) is defined on the domain:

    \[ D(f(A)) = \left\{ x \in H : \int_{\mathbb{R}} |f(\lambda)|^2 \, d\langle x, E_A(\lambda) x \rangle < \infty \right\}. \] ---

    2. Key Properties

    • Linearity: \( (a f + b g)(A) = a f(A) + b g(A)\) for scalars \(a, b\).
    • Multiplicative: \((f g)(A) = f(A) g(A)\) on the intersection of domains.
    • Adjoint: If \(f\) is complex-valued, \(f(A)^* = \overline{f}(A)\).
    • Norm: If \(f\) is bounded, \(\|f(A)\| = \sup_{\lambda \in \sigma(A)} |f(\lambda)|\).
    • Compatibility with polynomials: For \(f(\lambda) = \lambda^n\), \(f(A) = A^n\).
    ---

    3. Intuition

    • Think of \(f(A)\) as **applying \(f\) to the “eigenvalues”** of \(A\), even if \(A\) has a continuous spectrum.
    • It generalises diagonalisation: for bounded self-adjoint \(A\) with discrete spectrum \(\{\lambda_i\}\) and orthonormal eigenvectors \(e_i\), \[ f(A) = \sum_i f(\lambda_i) \, \langle \cdot, e_i \rangle e_i. \]
    • Allows constructing important operators like \(\exp(i t A)\) for unitary groups or \(|A|^{1/2}\) in quantum mechanics and PDEs.

    59: Unbounded Operators

    Question: Let \(A : D(A) \subset H \to H\) be a linear operator on a Hilbert space \(H\). Define what it means for \(A\) to be unbounded and prove that this is equivalent to saying no constant \(M\) exists such that \(\|A x\| \le M \|x\|\) for all \(x \in D(A)\).

    Solution / Proof

    Definition:

    A linear operator \(A : D(A) \subset H \to H\) is called bounded if there exists a constant \(M \ge 0\) such that

    \[ \|A x\| \le M \|x\|, \quad \forall x \in D(A). \]

    If no such \(M\) exists or \(D(A) \neq H\), the operator is called unbounded.

    ---

    Proposition:

    \(A\) is unbounded \(\iff\) for every \(n \in \mathbb{N}\), there exists \(x_n \in D(A)\) such that \(\|A x_n\| > n \|x_n\|\).

    ---

    Proof:

    1. (⇒) Suppose \(A\) is unbounded: By definition, no \(M\) exists such that \(\|A x\| \le M \|x\|\) for all \(x \in D(A)\). Therefore, for each \(n \in \mathbb{N}\), the inequality \(\|A x\| \le n \|x\|\) must fail for some \(x_n \in D(A)\). That is, \(\|A x_n\| > n \|x_n\|\).
    2. (⇐) Conversely, suppose for each \(n\) there exists \(x_n\) with \(\|A x_n\| > n \|x_n\|\): If \(A\) were bounded, there would exist \(M \ge 0\) such that \(\|A x\| \le M \|x\|\) for all \(x \in D(A)\). Choosing \(n > M\) gives a contradiction: \(\|A x_n\| > n \|x_n\| > M \|x_n\|\). Hence, \(A\) is unbounded.

    Conclusion: The two formulations are equivalent: unboundedness means no uniform bound exists on \(\|A x\| / \|x\|\) over its domain.

    60: Closed Operators

    Question: Let \(A : D(A) \subset H \to H\) be a linear operator on a Hilbert space \(H\). Define what it means for \(A\) to be closed and explain its significance.

    Solution / Notes

    Definition:

    An operator \(A : D(A) \subset H \to H\) is called closed if its graph

    \[ G(A) := \{ (x, A x) \in H \times H : x \in D(A) \} \] is a **closed subset** of \(H \times H\) with the product topology (norm on \(H \times H\) given by \(\|(x,y)\| = \sqrt{\|x\|^2 + \|y\|^2}\)).

    ---

    Equivalent characterisation:

    \(A\) is closed if and only if whenever a sequence \((x_n) \subset D(A)\) satisfies

    \[ x_n \to x \quad \text{and} \quad A x_n \to y \quad \text{in } H, \] then \(x \in D(A)\) and \(A x = y\).

    ---

    Intuition:

    • Think of a closed operator as one whose “limit points” of the graph remain inside the graph.
    • This ensures stability under limits: if a sequence of inputs converges and their images under \(A\) converge, then the limit is still in the domain and maps correctly.
    • Closedness is crucial for defining **unbounded operators** that behave like bounded ones under limits (e.g., differentiation operator in \(L^2\)).

    Self-Adjoint Operators

    Key Points:

    • An operator \(A : D(A) \subset H \to H\) is self-adjoint if:
      • \(A\) is symmetric: \(\langle Ax, y \rangle = \langle x, Ay \rangle\) for all \(x, y \in D(A)\).
      • \(A = A^*\), i.e., its adjoint \(A^*\) has the same domain and coincides with \(A\): \(D(A^*) = D(A)\) and \(A^*x = Ax\) for all \(x \in D(A)\).
    • Consequences:
      • Every self-adjoint operator is closed.
      • A symmetric operator is not necessarily self-adjoint; self-adjointness requires the domain to match the adjoint’s domain.
      • Self-adjointness is essential for spectral theorem, functional calculus, and defining unitary evolution in quantum mechanics.

    61: Stone’s Theorem (Strongly Continuous Unitary Groups)

    Key Points:

    • A family \(\{U(t)\}_{t \in \mathbb{R}}\) of operators on a Hilbert space \(H\) is a strongly continuous unitary group (SCUG) if:
      • \(U(0) = I\) (identity operator)
      • \(U(t+s) = U(t) U(s)\) for all \(s,t \in \mathbb{R}\) (group property)
      • \(t \mapsto U(t)x\) is continuous in the norm of \(H\) for all \(x \in H\) (strong continuity)
      • \(U(t)^* = U(-t)\) (unitarity)
    • Stone’s Theorem:
      • There exists a unique densely defined **self-adjoint operator \(A\)** on \(H\) such that:
        • \[ U(t) = e^{i t A}, \quad t \in \mathbb{R}. \]
      • Conversely, every self-adjoint operator \(A\) generates a SCUG via \(U(t) = e^{i t A}\).
    • Consequences / Intuition:
      • Provides a one-to-one correspondence between **self-adjoint operators** and **strongly continuous unitary evolution**.
      • Guarantees **unitary quantum evolution** in Hilbert space: time evolution is norm-preserving.
      • Operator \(A\) is interpreted as the **Hamiltonian** in quantum mechanics.

    62: Normed and Banach Algebras

    Definition:

    • A normed algebra \(A\) over \(\mathbb{C}\) is a vector space equipped with:
      • A bilinear product \(A \times A \to A, \ (x,y) \mapsto xy\)
      • A norm \(\|\cdot\|\) such that \(\|xy\| \le \|x\| \, \|y\|\) for all \(x,y \in A\)
    • If \(A\) is complete with respect to the norm, it is called a Banach algebra.
    • \(A\) is unital if there exists an element \(1 \in A\) (identity) such that:
      • \(1 \cdot x = x \cdot 1 = x\) for all \(x \in A\)
      • \(\|1\| = 1\)
    ---

    Example:

    • \(C([0,1])\), the space of continuous complex-valued functions on \([0,1]\), with:
      • Pointwise multiplication: \((f \cdot g)(x) = f(x) g(x)\)
      • Supremum norm: \(\|f\|_\infty = \sup_{x \in [0,1]} |f(x)|\)
      • This is a unital Banach algebra (identity is \(1(x) \equiv 1\)).

    63: Spectral Radius

    Question: Let \(A\) be a Banach algebra element. What is its spectral radius and how can it be expressed?

    Solution / Notes

    Definition:

    The spectral radius of an element \(a \in A\) is

    \[ r(a) := \sup \{ |\lambda| : \lambda \in \sigma(a) \}, \] where \(\sigma(a)\) is the spectrum of \(a\), i.e., all \(\lambda \in \mathbb{C}\) such that \(a - \lambda 1\) is not invertible in \(A\).

    ---

    Key Expression (Spectral Radius Formula):

    \[ r(a) = \lim_{n \to \infty} \|a^n\|^{1/n}. \]

    This gives a way to compute or estimate the spectral radius using powers of \(a\) in the Banach algebra.

    64: Maximal Ideals in Banach Algebras

    Question: What is a maximal ideal in a Banach algebra and what is the key theorem relating maximal ideals to homomorphisms?

    Solution / Notes

    Definition:

    Let \(A\) be a unital Banach algebra over \(\mathbb{C}\). An ideal \(M \subset A\) is called maximal if:

    • \(M \neq A\)
    • There is no proper ideal \(I\) such that \(M \subsetneq I \subsetneq A\)
    ---

    Key Theorem (Gelfand-Mazur / Maximal Ideal Correspondence):

    In a unital commutative Banach algebra \(A\):

    • Every maximal ideal \(M \subset A\) is the kernel of a non-zero **algebra homomorphism** \(\phi : A \to \mathbb{C}\).
    • Conversely, the kernel of any non-zero homomorphism \(\phi : A \to \mathbb{C}\) is a maximal ideal.
    ---

    Implications:

    • This establishes a **one-to-one correspondence** between maximal ideals and non-zero complex-valued characters (homomorphisms) on \(A\).
    • It is fundamental for **Gelfand theory**, which studies Banach algebras via their maximal ideals and spectra.

    65: Gelfand Theory: Commutative Banach Algebras

    Question: Let \(A\) be a unital commutative Banach algebra. Define the Gelfand transform, describe its properties, and explain the role of algebra homomorphisms.

    Solution / Notes

    1. Characters / Homomorphisms:

    • A character on \(A\) is a non-zero algebra homomorphism \(\phi : A \to \mathbb{C}\).
    • The set of all characters is called the maximal ideal space or Gelfand space \(\mathcal{M}(A)\).
    • Each character \(\phi\) corresponds to a maximal ideal \(M = \ker \phi\).
    ---

    2. Gelfand Transform:

    The Gelfand transform maps an element \(a \in A\) to a function \(\hat{a} : \mathcal{M}(A) \to \mathbb{C}\) defined by:

    \[ \hat{a}(\phi) := \phi(a), \quad \forall \phi \in \mathcal{M}(A). \]
    • \(\hat{a}\) is continuous with respect to the weak-* topology on \(\mathcal{M}(A)\).
    • The map \(a \mapsto \hat{a}\) is an algebra homomorphism: \(\widehat{ab} = \hat{a} \hat{b}\), \(\widehat{a+b} = \hat{a} + \hat{b}\), \(\widehat{\lambda a} = \lambda \hat{a}\).
    ---

    3. Properties / Significance:

    • The **spectrum** of \(a \in A\) satisfies: \(\sigma(a) = \{\phi(a) : \phi \in \mathcal{M}(A)\} = \hat{a}(\mathcal{M}(A))\).
    • The Gelfand transform is **norm-decreasing**: \(\|\hat{a}\|_\infty \le \|a\|\).
    • For **commutative C*-algebras**, the Gelfand transform is an isometric *-isomorphism onto \(C(\mathcal{M}(A))\).
    • Intuition: Gelfand theory allows one to study abstract Banach algebras via **continuous functions on the space of characters**.

    Gelfand-Mazur Theorem

    Question: State and prove the Gelfand-Mazur theorem for unital Banach algebras.

    Solution / Proof

    Theorem (Gelfand-Mazur):

    If \(A\) is a unital Banach algebra over \(\mathbb{C}\) in which every non-zero element is invertible, then \(A \cong \mathbb{C}\); that is, \(A\) is **isometrically isomorphic to the complex numbers**.

    ---

    Proof:

    1. Let \(a \in A\) be arbitrary. Since \(a \neq 0 \implies a\) is invertible, the spectrum \(\sigma(a)\) cannot contain \(0\).
    2. In a unital Banach algebra, the spectrum \(\sigma(a)\) is **non-empty** for any \(a \in A\) (standard spectral theorem for Banach algebras).
    3. Thus, \(\sigma(a)\) must consist of a **single non-zero complex number** \(\lambda \in \mathbb{C}\).
    4. Consider \(b := a - \lambda 1\). Then \(\sigma(b) = \sigma(a - \lambda 1) = \sigma(a) - \lambda = \{0\}\), so \(b\) is **not invertible**.
    5. But by assumption, **every non-zero element is invertible**, so \(b = 0 \implies a = \lambda 1\).
    6. Since \(a\) was arbitrary, every element in \(A\) is a scalar multiple of the identity. Therefore, \(A \cong \mathbb{C}\).

    Conclusion: A unital Banach algebra in which all non-zero elements are invertible is necessarily **one-dimensional** and isomorphic to \(\mathbb{C}\).

    Gelfand Representation Theorem

    Question: State the Gelfand representation theorem for a commutative unital Banach algebra and explain its significance.

    Solution / Notes

    Theorem (Gelfand Representation):

    Let \(A\) be a unital **commutative Banach algebra** over \(\mathbb{C}\). Denote by \(\mathcal{M}(A)\) the set of all non-zero algebra homomorphisms (characters) \(\phi: A \to \mathbb{C}\), equipped with the weak-* topology. Then the Gelfand transform:

    \[ \Gamma: A \to C(\mathcal{M}(A)), \quad a \mapsto \hat{a}, \quad \hat{a}(\phi) := \phi(a) \] satisfies:
    1. \(\Gamma\) is a **continuous algebra homomorphism**.
    2. \(\sigma(a) = \hat{a}(\mathcal{M}(A))\), so the spectrum of \(a\) is captured by the image of \(\hat{a}\).
    3. \(\|\hat{a}\|_\infty \le \|a\|\), and if \(A\) is a **commutative C*-algebra**, \(\Gamma\) is an **isometric *-isomorphism** onto \(C(\mathcal{M}(A))\).
    ---

    Significance:

    • The Gelfand representation allows one to study abstract commutative Banach algebras via **continuous functions on the space of characters**, turning algebraic problems into function-theoretic problems.
    • It provides a **concrete realisation** of elements in \(A\) as functions, linking the algebraic spectrum with the analytic image under \(\Gamma\).
    • This forms the foundation for **spectral theory** in commutative Banach algebras and for applications in **functional analysis and operator theory**.

    64: C*-Algebras and Key Consequences

    Question: What is a C*-algebra? State its defining properties and list the main consequences.

    Solution / Notes

    Definition:

    A C*-algebra \(A\) is a Banach algebra over \(\mathbb{C}\) equipped with an involution \( * : A \to A\) (conjugate-linear, \((ab)^* = b^* a^*, (a^*)^* = a\)) such that:

    1. \(\|a^* a\| = \|a\|^2\) for all \(a \in A\) (C*-identity).
    2. It is a Banach algebra: complete normed algebra with sub-multiplicative norm.
    ---

    Key Consequences:

    • Gelfand-Naimark Theorem: Every commutative unital C*-algebra \(A\) is *-isomorphic to \(C(X)\), the algebra of continuous complex-valued functions on a compact Hausdorff space \(X\) (its spectrum / character space).
    • Self-adjoint elements: Elements \(a = a^*\) have **real spectrum**, i.e., \(\sigma(a) \subset \mathbb{R}\).
    • Norm via spectrum: For any \(a \in A\), \(\|a\| = r(a)\) if \(a\) is normal (\(aa^* = a^* a\)), so the spectral radius equals the operator norm.
    • Functional calculus: For normal elements \(a \in A\), any continuous function \(f\) on \(\sigma(a)\) defines an element \(f(a) \in A\), extending polynomials to continuous functions.
    • Representation on Hilbert space: Every C*-algebra can be represented as a norm-closed *-subalgebra of \(B(H)\), the bounded operators on some Hilbert space \(H\) (Gelfand-Naimark-Segal construction).
    ---

    Intuition:

    C*-algebras generalise the algebra of bounded operators on Hilbert space while keeping **norm, involution, and spectral properties tightly linked**. They are central to quantum mechanics, operator theory, and non-commutative geometry.

    65: Representations of Algebras

    Question: What is a representation of a (Banach or C*)-algebra?

    Solution / Notes

    Definition:

    A representation of an algebra \(A\) on a Hilbert space \(H\) is a **homomorphism**:

    \[ \pi : A \to B(H) \] such that for all \(a, b \in A\) and \(\lambda \in \mathbb{C}\):
    1. \(\pi(a + b) = \pi(a) + \pi(b)\), \(\pi(\lambda a) = \lambda \pi(a)\)
    2. \(\pi(ab) = \pi(a) \pi(b)\)
    3. For a *-algebra, \(\pi(a^*) = \pi(a)^*\) (preserves involution)

    Intuition: A representation realises abstract algebra elements as **concrete bounded operators on a Hilbert space**.

    66: GNS Construction (Gelfand–Naimark–Segal)

    Question: Explain the GNS construction for a C*-algebra \(A\) and a positive linear functional \(\phi\).

    Solution / Notes

    Setup:

    Let \(A\) be a C*-algebra and \(\phi: A \to \mathbb{C}\) a **positive linear functional** (\(\phi(a^* a) \ge 0\) for all \(a\in A\)).

    ---

    Construction:

    1. Define a semi-inner product on \(A\) by \(\langle a, b \rangle := \phi(b^* a)\).
    2. Define the **null space** \(N := \{ a \in A : \phi(a^* a) = 0 \}\).
    3. Form the quotient space \(A / N\) and let \([a]\) denote the equivalence class of \(a\). Then \(\langle [a], [b] \rangle := \phi(b^* a)\) is a genuine inner product.
    4. Complete \(A / N\) with respect to this inner product to get a **Hilbert space** \(H_\phi\).
    5. Define the representation \(\pi_\phi: A \to B(H_\phi)\) by \(\pi_\phi(a) [b] := [ab]\). This is a ***-representation**.
    6. The vector \(\xi_\phi := [1_A]\) (if \(A\) is unital) is **cyclic**, meaning \(\pi_\phi(A) \xi_\phi\) is dense in \(H_\phi\), and \(\phi(a) = \langle \pi_\phi(a) \xi_\phi, \xi_\phi \rangle\).
    ---

    Key Consequences:

    • Every C*-algebra has a faithful representation as operators on some Hilbert space (via direct sums of GNS representations).
    • Provides a **bridge between abstract C*-algebras and concrete Hilbert space operators**.
    • Fundamental in quantum mechanics: positive functionals → states → operator representation.

    67: Gelfand–Naimark Theorem

    Question: State and prove the Gelfand–Naimark theorem for C*-algebras.

    Solution / Notes

    Theorem (Gelfand–Naimark):

    Every C*-algebra \(A\) is *-isomorphic to a norm-closed *-subalgebra of \(B(H)\), the bounded operators on some Hilbert space \(H\). In other words, every abstract C*-algebra can be represented concretely as operators on a Hilbert space.

    ---

    Proof Outline:

    1. Let \(A\) be a C*-algebra. Consider the set of all **positive linear functionals** \(\phi: A \to \mathbb{C}\) (states).
    2. For each \(\phi\), construct the **GNS representation** \((\pi_\phi, H_\phi, \xi_\phi)\) as described in the previous section.
    3. Take the **direct sum** of all GNS representations: \[ \pi := \bigoplus_\phi \pi_\phi : A \to B\Big(\bigoplus_\phi H_\phi\Big) \] This is a *-representation of \(A\) on a (possibly large) Hilbert space.
    4. \(\pi\) is **injective (faithful)**: if \(a \neq 0\), there exists a positive linear functional \(\phi\) such that \(\phi(a^* a) > 0\), hence \(\pi_\phi(a) \neq 0\). Therefore, \(\pi(a) \neq 0\).
    5. The image \(\pi(A)\) is a **norm-closed *-subalgebra** of \(B(\bigoplus_\phi H_\phi)\), and \(\pi\) preserves the *-algebra structure and the norm (\(\|\pi(a)\| = \|a\|\)).

    Conclusion:

    Thus, any abstract C*-algebra can be faithfully realised as a concrete algebra of bounded operators on a Hilbert space. This establishes the **Gelfand–Naimark representation theorem**.

    68: Non-linear Functional Analysis

    Fixed Point Theorems

    Banach Fixed Point Theorem (Contraction Mapping)

    Question: State the Banach fixed point theorem and prove it.

    Solution / Notes

    Theorem (Banach Fixed Point / Contraction Mapping):

    Let \((X, d)\) be a **complete metric space** and \(T: X \to X\) a **contraction**, i.e., there exists \(0 \le k < 1\) such that:

    \[ d(T(x), T(y)) \le k \, d(x, y) \quad \forall x, y \in X. \] Then:

    1. There exists a **unique fixed point** \(x^* \in X\) such that \(T(x^*) = x^*\).
    2. For any \(x_0 \in X\), the sequence \(x_{n+1} = T(x_n)\) converges to \(x^*\) with estimate: \[ d(x_n, x^*) \le \frac{k^n}{1-k} d(x_1, x_0). \]
    ---

    Proof:

    1. Construct the iterative sequence: Choose \(x_0 \in X\), define \(x_{n+1} = T(x_n)\).
    2. Show Cauchy property: For \(n > m\): \[ d(x_n, x_m) \le d(x_n, x_{n-1}) + \dots + d(x_{m+1}, x_m) \le d(x_1, x_0) \sum_{j=m}^{n-1} k^j \le \frac{k^m}{1-k} d(x_1, x_0). \] As \(m \to \infty\), \(d(x_n, x_m) \to 0\). Hence \((x_n)\) is **Cauchy**.
    3. Completeness: Since \(X\) is complete, \(x_n \to x^* \in X\).
    4. Fixed point property: By continuity of \(T\) (follows from contraction): \[ T(x^*) = T\Big(\lim_{n \to \infty} x_n\Big) = \lim_{n \to \infty} T(x_n) = \lim_{n \to \infty} x_{n+1} = x^*. \]
    5. Uniqueness: Suppose \(y^*\) is another fixed point: \[ d(x^*, y^*) = d(T(x^*), T(y^*)) \le k \, d(x^*, y^*). \] Since \(0 \le k < 1\), this implies \(d(x^*, y^*) = 0 \Rightarrow x^* = y^*\).

    Conclusion:

    Every contraction on a complete metric space has a **unique fixed point**, and iterating the map converges to it. This theorem is fundamental in proving **existence and uniqueness** in ODEs, integral equations, and functional analysis.

    Schauder Fixed Point Theorem

    Question: State and prove the Schauder fixed point theorem.

    Solution / Notes

    Theorem (Schauder Fixed Point):

    Let \(C\) be a **non-empty, convex, compact subset** of a Banach space \(X\). Let \(T: C \to C\) be **continuous**. Then \(T\) has at least **one fixed point** in \(C\), i.e., there exists \(x^* \in C\) such that:

    \[ T(x^*) = x^*. \] ---

    Proof Outline:

    1. Finite-dimensional approximation: By compactness, for any \(n\) there exists a finite-dimensional subspace \(X_n \subset X\) and a continuous map \(P_n: C \to X_n \cap C\) that approximates the identity (projection onto finite-dimensional slices).
    2. Apply Brouwer in finite dimension: \(T_n := P_n \circ T: X_n \cap C \to X_n \cap C\) is continuous on a convex, compact subset of \(\mathbb{R}^n\). By **Brouwer’s fixed point theorem**, \(T_n\) has a fixed point \(x_n \in X_n \cap C\).
    3. Extract convergent subsequence: Since \(C\) is compact, the sequence \((x_n)\) has a convergent subsequence \(x_{n_k} \to x^* \in C\).
    4. Pass to the limit: Continuity of \(T\) gives: \[ T(x^*) = \lim_{k \to \infty} T(x_{n_k}) = \lim_{k \to \infty} x_{n_k} = x^*. \]

    Conclusion:

    Schauder’s theorem generalises **Brouwer’s fixed point theorem** to infinite-dimensional Banach spaces under compactness and convexity. It is widely used for proving existence of solutions in **integral equations, PDEs, and nonlinear functional analysis**.

    69: Variational Methods

    1. Weak Formulation

    Question: Explain what the weak formulation of a boundary value problem is.

    Solution / Notes

    Weak Formulation:

    • Start with a differential equation (e.g., boundary value problem) of the form \(-\Delta u = f\) in \(\Omega\), \(u = 0\) on \(\partial \Omega\).
    • Multiply by a **test function** \(v\) from a suitable space (e.g., \(H_0^1(\Omega)\)) and integrate over the domain \(\Omega\): \[ \int_\Omega -\Delta u \, v \, dx = \int_\Omega f v \, dx. \]
    • Apply **integration by parts** to move derivatives off \(u\) (reduces smoothness requirements): \[ \int_\Omega \nabla u \cdot \nabla v \, dx = \int_\Omega f v \, dx. \]
    • Find \(u \in H_0^1(\Omega)\) satisfying this for all test functions \(v \in H_0^1(\Omega)\). This is the **weak formulation**.

    Key point: Weak formulation allows solutions with less regularity, which is essential in functional analysis and PDE theory.

    2. Direct Method of Calculus of Variations

    Question: State the direct method in the calculus of variations and its main theorem.

    Solution / Notes

    Direct Method (Calculus of Variations):

    Let \(X\) be a reflexive Banach space, and let \(J: X \to \mathbb{R} \cup \{+\infty\}\) be a functional. Suppose:

    1. \(J\) is **coercive**: \(\|u\|_X \to \infty \implies J(u) \to +\infty\).
    2. \(J\) is **weakly lower semi-continuous**: \(u_n \rightharpoonup u \implies J(u) \le \liminf J(u_n)\).

    Then **\(J\) attains its minimum**: there exists \(u^* \in X\) such that \[ J(u^*) = \inf_{u \in X} J(u). \]

    Proof Outline:

    1. Take a minimising sequence \((u_n)\) such that \(J(u_n) \to \inf J\).
    2. Coercivity ensures \((u_n)\) is **bounded** in \(X\).
    3. Reflexivity of \(X\) implies there exists a **weakly convergent subsequence** \(u_{n_k} \rightharpoonup u^*\).
    4. Weak lower semi-continuity gives: \[ J(u^*) \le \liminf_{k \to \infty} J(u_{n_k}) = \inf J, \] hence \(u^*\) is a minimiser.

    Key Point:

    The direct method provides an **existence result** for minimisers in variational problems using functional-analytic properties, without requiring explicit solutions.

    3. Euler–Lagrange Equations

    Question: Derive the Euler–Lagrange equation for a functional \(J(u) = \int_\Omega F(x, u, \nabla u)\, dx\).

    Solution / Notes

    Euler–Lagrange Equations:

    1. Consider a functional \(J: X \to \mathbb{R}\), \(X\) a suitable function space (e.g., \(H^1(\Omega)\)), \[ J(u) = \int_\Omega F(x, u(x), \nabla u(x)) \, dx. \]
    2. Take a variation \(v \in X\), define \(u_\epsilon = u + \epsilon v\). Compute \[ \frac{d}{d\epsilon} J(u_\epsilon) \Big|_{\epsilon=0} = 0 \quad \text{(necessary condition for extremum).} \]
    3. Using integration by parts and assuming smooth boundary conditions, the **Euler–Lagrange equation** is obtained: \[ \frac{\partial F}{\partial u} - \sum_{i=1}^n \frac{d}{dx_i} \frac{\partial F}{\partial (\partial u / \partial x_i)} = 0 \quad \text{in } \Omega. \]

    Key point: E-L equations give the **strong form** corresponding to a variational problem.

    4. Weak Compactness Method

    Question: Explain the weak compactness method in variational problems and how it is used to obtain solutions.

    Solution / Notes

    Weak Compactness Method:

    • Used in **infinite-dimensional variational problems** where direct minimisation may fail in norm topology.
    • Key idea: If \(X\) is a **reflexive Banach space**, every bounded sequence has a **weakly convergent subsequence**.
    • Combine **weak convergence + weak lower semi-continuity** of the functional to extract a **minimiser** in the weak sense.

    It is widely used in **nonlinear PDEs** where compactness in norm is unavailable.

    5. Rellich–Kondrachov Compactness and Concentration-Compactness Principle

    Question: State the Rellich–Kondrachov theorem and explain the concentration-compactness principle.

    Solution / Notes

    Rellich–Kondrachov Compactness Theorem:

    Let \(\Omega \subset \mathbb{R}^n\) be bounded with Lipschitz boundary. For \(1 \le p < \infty\) and \(1 \le q < p^*\) (Sobolev conjugate):

    \[ H_0^1(\Omega) \hookrightarrow L^q(\Omega) \quad \text{is **compact**.} \]

    Consequences:

    • Every bounded sequence in \(H_0^1(\Omega)\) has a **strongly convergent subsequence in \(L^q(\Omega)\)**.
    • Used to pass to the limit in **nonlinear functionals** in variational methods.

    Concentration-Compactness Principle (P.-L. Lions):

    Addresses **lack of compactness** due to translation or dilation in unbounded domains:

    • Given a bounded sequence in a function space, one of three phenomena occurs:
      1. Compactness: subsequence strongly converges.
      2. Vanishing: mass spreads out, no limit.
      3. Concentration: mass concentrates at points, possibly forming Dirac-type measures.
    • Allows **recovery of convergence** by understanding the limiting behavior of sequences that would otherwise "escape" compactness.

    Applications: **Nonlinear PDEs, critical Sobolev exponents, blow-up analysis**.
